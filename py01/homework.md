> **제출 기한**: 다음 수업 전날 23:59
> **분량**: A4 0.5페이지 내외 (이미지 포함 가능)

## 과제 1: 강사에게 자기소개 이메일 보내기

비즈니스 이메일 원칙에 따라 강사에게 자기소개 이메일을 보냅니다.

**수신**: [강사 이메일 주소 — 강의실 화이트보드 확인]

**이메일에 반드시 포함해야 할 내용**

| 항목 | 내용 |
|:---|:---|
| 제목 | `[학번 이름] 자기소개 메일` 형식 — 예: `[20251234 홍길동] 자기소개 메일` |
| 인사 | 수신자에게 짧은 인사 |
| 본문 첫 문장 | "자기소개 메일 드립니다" (두괄식) |
| 자기소개 | 이름, 학번, 이 수업에 기대하는 것 한 줄 |
| 끝맺음 | 이름 + 소속 |

**체크리스트** (이메일 보내기 전 확인)
- [ ] 첨부 파일이 필요한 경우 먼저 추가했는가?
- [ ] 제목만 읽어도 내용을 알 수 있는가?
- [ ] 본문 첫 문장에 요점이 나오는가?
- [ ] 맞춤법에 오류가 없는가?
- [ ] To(받는 사람)에 강사 이메일이 정확히 입력됐는가?

## 과제 2: 계정 설정 완료 인증

아래 3가지 화면을 캡처하여 eCampus 제출함에 함께 제출합니다.

**인증 1: Google Colab 노트북 실행 화면**
- `print("안녕하세요!")` 코드와 실행 결과가 보이는 화면
- 브라우저 주소창(colab.research.google.com)이 보여야 합니다.

**인증 2: GitHub 프로필 페이지**
- `github.com/본인아이디` 주소가 브라우저에 표시된 상태의 화면
- 프로필 이름이 설정된 상태여야 합니다.

**인증 3: AI 도구 대화 화면**
- ChatGPT 또는 Gemini에서 "파이썬이 뭐야?"를 질문한 결과 화면
- AI 도구 이름(ChatGPT/Gemini)이 화면에 보여야 합니다.

## 참고: 제출 파일 구성 예시

eCampus 제출 시 아래 두 가지 중 하나로 제출합니다.

**방법 A**: A4 문서 1장에 아래 내용을 담아 PDF로 제출
- 이메일 보낸 화면 캡처
- 인증 1, 2, 3 화면 캡처
- 간단한 소감 (한 줄)

**방법 B**: eCampus 텍스트 제출함에 소감 입력 + 이미지 파일 별도 첨부
