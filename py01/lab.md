## 실습 개요
본 실습은 별도의 프로그램 설치 없이 웹 브라우저 위에서 파이썬 코드를 작성하고 실행하는 방법을 배웁니다. `print()` 함수를 활용하여 텍스트를 출력해 보고, 고의로 명령어에 실수를 만들어 에러 메시지를 확인하는 훈련을 합니다.

## Step 1: Google Colab 접속 및 준비
1. **Google 계정 로그인**: 본인의 구글 이메일 환경에 로그인합니다.
2. **Colab 접속**: 웹 브라우저 검색창에 `Google Colab`을 검색하거나, `colab.research.google.com` 주소로 접속합니다.
3. **새 노트 만들기**: 
   - 처음 화면 창에서 [새 노트] 버튼을 클릭합니다.
   - 나타나는 빈 화면이 우리가 요리를 할 '주방(도마)'입니다.
   - 상단의 파일 이름(기본값: `Untitled0.ipynb`)을 클릭하여 `week2_practice.ipynb`로 변경해 봅시다.

## Step 2: `print()` 함수 다루기
1. 화면에 보이는 직사각형 칸(코드 셀)을 클릭합니다.
2. 아래의 코드를 똑같이 타이핑합니다. (대소문자와 괄호, 따옴표에 주의하세요.)
```python
print("Hello, Welcome to Python!")
print("오늘 서울의 날씨는 맑음입니다.")
```
3. 코드 셀 왼쪽의 **재생(Play) 버튼**을 누르거나, 키보드의 `Shift + Enter`를 누릅니다.
4. 아래쪽에 코드가 실행된 결과가 출력되는 것을 확인합니다.

## Step 3: 숫자와 문자의 조합
1. 아래 코드를 입력하고 실행해 보세요.
```python

print(2 + 1)
print("2 + 1")
```
2. 따옴표가 없는 숫자의 계산 결과(`3`)와 따옴표가 있는 문자열(`2 + 1`) 출력 결과의 차이를 눈으로 확인합니다.
