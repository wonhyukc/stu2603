## 실습 1: 개발 환경 체험 - Google Colab (구글 코랩)

본인의 노트북에 파이썬을 복잡하게 설치할 필요가 없습니다. 구글 계정과 웹 브라우저(크롬 권장)만 있으면 구글의 클라우드 컴퓨터를 빌려서 바로 코딩할 수 있습니다.

### [진행 순서]
1. 구글 크롬 브라우저를 열고 개인 구글 계정으로 로그인합니다.
2. 주소창에 `colab.research.google.com`을 입력하여 접속합니다.
3. **[파일] -> [새 노트]** 버튼을 클릭하여 새로운 파이썬 코딩 환경 파일을 만듭니다.
4. 첫 화면의 빈칸(코드 셀) 부분에 아래 내용을 타이핑해 봅니다.
   ```python
   print("Hello, Seoul Theological University!")
   ```
5. 좌측에 있는 `재생 버튼(▶)`을 누르거나, 키보드의 `Shift + Enter`를 누릅니다.
6. 코드 아래에 정상적으로 텍스트가 출력되는지 확인합니다. 축하합니다! 여러분의 생애 첫 파이썬 코드가 클라우드에서 실행되었습니다.

## 실습 2: AI 자동 생성 도구 가볍게 체험하기

우리의 수업 파트너가 될 생성형 AI(ChatGPT, Claude, Gemini 중 택 1)에게 첫 인사를 건네고, 코드를 만들어달라고 부탁해 봅니다.

### [진행 순서]
1. 본인이 사용 가능한 AI 서비스(예: chatgpt.com)에 로그인합니다.
2. 아래와 같이 프롬프트(명령어)를 입력합니다.
   > "안녕, 나는 프로그래밍을 처음 배우는 1학년 대학생이야. 화면에 '나의 첫 AI 파트너, 반가워!'라고 출력하는 아주 간단한 파이썬 코드를 한 줄만 작성해 줘."
3. AI가 생성해준 코드를 복사합니다.
4. 앞서 띄워둔 Google Colab의 새로운 코드 셀에 복사한 코드를 붙여넣기(`Ctrl + V`) 합니다.
5. 실행(`Shift + Enter`) 버튼을 누르고 화면에 올바른 인사가 출력되는 것을 확인합니다.
