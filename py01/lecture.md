# [1주차] 오리엔테이션 및 컴퓨팅적 사고 입문 강의안

## 1. 수업 개요 (OT)
* **목표:** '컴퓨팅적사고와코딩(K트랙)' 과정의 목적과 진행 방식(거꾸로 수업)을 명확히 이해한다.
* **핵심 철학:** 프로그래밍 문법의 단순 암기가 아닌, **"AI를 파트너로 활용한 문제 해결력(AI-Native 컴퓨팅 사고)"** 배양.

## 2. 수업 운영 방식: 거꾸로 수업 (Flipped Learning)
* **사전 학습(온라인):** 매주 지정된 영상 강의(eCampus)를 통해 핵심 이론을 먼저 학습하고 옵니다.
* **대면 본수업(오프라인):** 강의실에서는 문법 설명 대신, 실제로 코드를 실습하고, AI가 작성한 코드의 오류를 찾는(코드 비평) 훈련에 집중합니다.

## 3. 프로페셔널 소통 가이드
* 우리는 오늘부터 단순한 학생이 아니라 **주니어 엔지니어**입니다.
* **게시판 질문 규칙 (엔지니어식 질문법):**
  1. **상황 (Situation):** 무엇을 만들거나 실행하려고 했는가?
  2. **시도 내역 (Attempts):** 문제를 해결하기 위해 어떤 시도를 해보았는가?
  3. **에러 메시지 (Error Log):** 화면에 출력된 에러 메시지 원문 (또는 화면 캡쳐).
  4. **AI의 답변 (AI Response):** 발생한 에러를 AI(ChatGPT, Claude 등)에게 물어보았을 때 어떤 답변을 받았는가?
* **이메일 원칙:** 명확한 제목, 소속(학번/이름), 결론부터 말하는 비즈니스 이메일 양식을 준수합니다.

## 4. 파이썬(Python) 코딩이란 무엇인가? (30% Intro)
* 파이썬은 사람의 언어(자연어)와 매우 유사하여 입문자가 배우기 가장 좋은 프로그래밍 언어입니다.
* 복잡한 환경 설정 없이 웹 브라우저만 있으면 곧바로 코딩을 시작할 수 있습니다.
* **이번 주 목표:** 복잡한 이론 구문은 생략하고, AI 번역기(생성형 AI)를 통해 간단한 파이썬 코드가 어떻게 만들어지는지 맛보기만 진행합니다.
