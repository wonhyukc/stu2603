# 4주차 파이썬 강의 가이드: 조건문 (if-elif-else)

## 주차 개요
* **학습 목표**: 
  1. `if-elif-else` 다중 분기 구조와 들여쓰기 규칙을 이해한다.
  2. 비교 연산자 6종 및 논리 연산자 3종을 조합하여 복합 조건식을 구성한다.
  3. 실생활 문제를 프로그래밍 분기 로직으로 전환하는 사고를 익힌다.
* **소요 시간**: 총 75분 (25분 x 3개 세션)

---

## 세션별 구성안

### [S1] `if-elif-else` 다중 분기 구조와 비교 연산자 (25분)
* **목표**: 조건문의 기본 원리, Python 특유의 들여쓰기(Indentation) 규칙 이해, 비교 연산자 활용.
* **주요 개념**:
  * 단일 제어 (`if`), 이중 제어 (`if-else`), 다중 제어 (`if-elif-else`) 흐름 비교.
  * 갈림길 비유: 내비게이션 경로 선택.
  * 파이썬 들여쓰기 규칙 (공백 4칸).
  * 비교 연산자 6종: `==`, `!=`, `>`, `<`, `>=`, `<=`.
* **강사 액션**:
  * 콘솔을 띄운 상태에서 `score = 85`을 주고 `if score >= 80:` 조건이 통과하여 `print("합격")`이 출력되는 라이브 코딩.

### [S2] 논리 연산자와 중첩 조건문 (25분)
* **목표**: 복합적인 조건을 조합하는 방법과 논리 연산자(`and`, `or`, `not`)의 의미 이해.
* **주요 개념**:
  * `and`: 두 개의 문(조건)을 모두 통과해야 열리는 이중 철문 비유.
  * `or`: 두 개의 카드 중 하나만 제출해도 통과되는 패스 비유.
  * `not`: 청개구리 변환. True를 False로, False를 True로.
  * 중첩 `if`문 (Condition 안의 Condition).
* **강사 액션**:
  * 진리표(Truth Table)를 직관적으로 설명하며, 나이와 VIP 회원 여부를 조합하여 놀이공원 입장료를 면제해주는 간단한 조건 분기 작성.

### [S3] 라이브 코딩: 학점 계산기 만들기 및 AI 코드 비평 (25분)
* **목표**: 배운 내용을 총망라하여 실습 프로그램을 구현하고, AI를 활용해 로직을 고도화하는 경험.
* **주요 개념**:
  * `input()` 함수로 사용자로부터 점수를 입력받아 정수인 `int()`로 형변환.
  * `if-elif-else`를 통한 90, 80, 70점 등급(`A`, `B`, `C`, `F`) 출력 프로그래밍.
  * AI 챗봇을 활용해 "점수가 0~100 사이가 아닌 경우의 예외 처리"를 물어보고 코드를 강화(Critique)하는 시연.
* **강사 액션**:
  * 백지 상태에서 코드를 차근차근 입력.
  * 의도적으로 에러를 내고 (문자열 대소 비교 등) 에러 메시지를 AI에게 물어보는 과정 시연.
  * 학생들에게 `handout.md`에 나온 메일 과제를 명확하게 안내 (일상의 결정 트리 작성).
