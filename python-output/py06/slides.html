💡 외부 데이터 가져오기 3가지 기술

웹 표 복사

Ctrl+Shift+V (값만 붙여넣기)를 통해 웹의 지저분한 서식을 제거하고 순수 데이터만 추출합니다.

CSV 업로드

[파일]-[가져오기]-[업로드]를 통해 쉼표로 구분된 데이터를 마법처럼 분할하여 로드합니다.

엑셀 호환

.xlsx 파일을 구글 드라이브에 올린 뒤 [Google Sheets로 열기]로 동시 협업 환경을 구축합니다.

🏗️ 뼈대 잡기: 틀 고정

방대한 데이터의 길잡이

데이터의 제목(Header)이 되는 1행을 고정하여 스크롤 중에도 항목 이름을 유지합니다.

  • [보기] - [틀 고정] - [행 1개]
  • 항목 이름을 잃지 않고 분석 가능

[틀 고정 설정 화면 예시]

🔍 시각적 정렬 1단계: 정렬(Sort)

오름차순 (A-Z)

작은 값에서 큰 값 순으로 정렬합니다.

내림차순 (Z-A)

큰 값에서 작은 값 순으로 정렬합니다.

순위표 완성

특정 열을 클릭하여 수초 만에 무작위 데이터를 재배열합니다.

🎨 시각적 정렬 2단계: 조건부 서식

색으로 말하는 데이터

수치의 높고 낮음에 따라 열화상 카메라처럼 그라데이션을 적용합니다.

  • [서식] - [조건부 서식] - [색상 스케일]
  • 코딩 없이 데이터 시각화 구현

[조건부 서식 색상 스케일 예시]

✨ 스마트 전처리: 데이터 분리 마법

스마트 자동 채우기

원하는 패턴을 시트가 파악하여 수백 명의 데이터를 한 번에 분리합니다.

예: Name 열에서 'Mr.', 'Miss.' 호칭만 분리할 때 직접 3번 정도 적으면 패턴을 파악하여 회색 미리보기를 제안합니다. (Enter!)
