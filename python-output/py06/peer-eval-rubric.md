# 6주차 실습 동료평가 기준표 (Peer Evaluation Rubric)

동료가 eCampus에 제출한 **[2가지 공유 링크(AI 교안 & 구글 시트)]**에 접속하여 아래 8가지 항목에 대해 객관적으로 채점(Yes / No)해 주시기 바랍니다. (어느 하나라도 불충분하면 단호하게 No(0)를 부여하세요.)

| 평가 항목 | 채점 기준 | 배점 |
| :--- | :--- | :--- |
| **Q1. AI 복습 교안 제출 여부** | 6주차 핵심 키워드를 바탕으로 AI(Gemini 등)가 작성한 **'맞춤형 복습 교안 대화창 링크'**가 구글 폼/eCampus에 제출되었습니까? | Yes(1) / No(0) |
| **Q2. AI 복습 교안 접근 가능성** | 제출된 AI 교안 링크를 클릭했을 때, 권한 막힘 없이 정상적인 화면으로 열립니까? | Yes(1) / No(0) |
| **Q3. 데이터 정상 가져오기** | 제출된 구글 시트에 타이타닉 탑승객 데이터(csv)가 온전히 불러와져 하나의 시트에 정리되어 있습니까? | Yes(1) / No(0) |
| **Q4. 호칭 추출 분리** | `Name` 컬럼 옆에 **'호칭' 전용 열이 새롭게 추가되어 데이터가 모두 채워져 분리**되어 있습니까? | Yes(1) / No(0) |
| **Q5. 총 승선객 수 세기 (피벗 활용)** | 분석용 피벗 테이블을 통해 타이타닉의 **'총 탑승객 수(개수)'** 값이 화면에 도출되어 있습니까? | Yes(1) / No(0) |
| **Q6. 평균 나이 도출 (피벗 활용)** | 동일한 피벗 테이블 내에 나이(`Age`) 데이터를 사용하여 **전체 평균 나이**가 정상적으로 분석되어 있습니까? | Yes(1) / No(0) |
| **Q7. 중위 나이 도출 (피벗 활용)** | 동일한 피벗 테이블 내에 나이(`Age`) 데이터를 사용하여 **전체 중위 나이(중앙값)**가 정상적으로 분석되어 있습니까? | Yes(1) / No(0) |
| **Q8. 생존 결과 교차 집계 (피벗 활용)** | 피벗 테이블 행(Row)에 성별이나 객실 등급을 넣고, 값(Values)에 생존 여부를 넣어 **다차원적인 교차 집계표**가 완성되어 있습니까? | Yes(1) / No(0) |

* 총점: 8점 만점 (각 문항 1점)
* **평가 시 주의사항**: 동료가 단순 텍스트 타이핑이나 캡처가 아닌, **'직접 접속 가능한 2개의 웹 링크'**를 올바르게 제출했는지 확인해야 합니다. 동료가 마법(스마트 채우기)이나 피벗 테이블 설정을 올바르게 다뤘는지 시트 곳곳을 클릭해 점검해 보세요.
