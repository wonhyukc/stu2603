# 6주차 실습 동료평가 기준표 (Peer Evaluation Rubric)

동료가 eCampus에 제출한 **[2가지 공유 링크(AI 교안 & 구글 시트)]**에 접속하여 아래 6가지 항목에 대해 객관적으로 채점(Yes / No)해 주시기 바랍니다.

> **[🔥 필수 통과 조건 (Fail-Fast)]**
> 제출된 2개의 링크(AI 교안, 구글 시트) 중 하나라도 미제출되었거나 권한 문제로 열리지 않을 경우, **아래 문항을 채점하지 말고 즉시 총점 0점을 부여하고 평가를 종료**합니다.

| 평가 항목 | 채점 기준 | 배점 |
| :--- | :--- | :--- |
| **Q1. 데이터 정상 가져오기** | 기초 데이터(csv)가 온전히 불러와져 하나의 시트에 정리/분할되어 있습니까? | Yes(1) / No(0) |
| **Q2. 첫 행 틀 고정 확인** | 스크롤 시에도 1행의 제목이 사라지지 않고 항상 상단에 유지되도록 '틀 고정' 처리가 되어 있습니까? | Yes(1) / No(0) |
| **Q3. 데이터 정렬 확인** | 데이터 열 중 하나가 오름차순 혹은 내림차순으로 한 번이라도 재정렬된 흔적이 시트에 나타나 있습니까? | Yes(1) / No(0) |
| **Q4. 조건부 서식(색상 스케일)** | 수치 값에 따라 색상 톤이 변하는 '색상 스케일 조건부 서식'이 시트 내에 올바르게 적용되어 있습니까? | Yes(1) / No(0) |
| **Q5. 호칭 추출 분리** | `Name` 옆 전용 열에 '스마트 채우기'로 도출한 호칭('Mr.', 'Mrs.' 등) 데이터가 400개 이상 새롭게 채워져 있습니까? | Yes(1) / No(0) |
| **Q6. 승선객 총숫자 (피벗)** | 분석용 피벗 테이블 공간에 총 탑승객 수(개수 집계) 값이 도출되어 있습니까? | Yes(1) / No(0) |
| **Q7. 평균 나이 (피벗)** | 동일한 피벗 테이블 내에 나이(`Age`) 데이터를 사용하여 전체 평균 나이가 정상 분석되어 있습니까? | Yes(1) / No(0) |
| **Q8. 중위 나이 (피벗)** | 동일한 피벗 테이블 내에 나이(`Age`) 데이터를 사용하여 전체 중위 나이(중앙값)가 정상 분석되어 있습니까? | Yes(1) / No(0) |
| **Q9. 교차 집계 완성 (피벗)** | 피벗 테이블의 행에 성별/등급을, 값에 생존결과를 매핑해 다차원 교차 집계표(피벗 테이블)가 만들어져 있습니까? | Yes(1) / No(0) |

* 총점: 9점 만점 (각 문항 1점)
* **평가 시 주의사항**: 구글 시트 곳곳을 클릭하여, 동료가 단순 텍스트 캡처나 타이핑을 한 것이 아니라 실제로 마법(스마트 채우기)과 피벗 테이블 기능을 능숙하게 다뤘는지 점검해 보세요.
