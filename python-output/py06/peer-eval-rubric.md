# 6주차 실습 동료평가 기준표 (Peer Evaluation Rubric)

동료가 eCampus에 제출한 **[2가지 공유 링크(AI 교안 & 구글 시트)]**에 접속하여 아래 6가지 항목에 대해 객관적으로 채점(Yes / No)해 주시기 바랍니다.

> **[🔥 필수 통과 조건 (Fail-Fast)]**
> 제출된 2개의 링크(AI 교안, 구글 시트) 중 하나라도 미제출되었거나 권한 문제로 열리지 않을 경우, **아래 문항을 채점하지 말고 즉시 총점 0점을 부여하고 평가를 종료**합니다.

| 평가 항목 | 채점 기준 | 배점 |
| :--- | :--- | :--- |
| **Q1. 데이터 정상 가져오기** | 타이타닉 탑승객 데이터(csv)가 온전히 불러와져 하나의 시트에 정리되어 있습니까? | Yes(1) / No(0) |
| **Q2. 호칭 추출 분리** | `Name` 컬럼 옆에 **'호칭' 전용 열이 새롭게 추가되어 데이터가 모두 채워져 분리**되어 있습니까? | Yes(1) / No(0) |
| **Q3. 총 승선객 수 세기 (피벗 활용)** | 분석용 피벗 테이블을 통해 타이타닉의 **'총 탑승객 수(개수)'** 값이 화면에 도출되어 있습니까? | Yes(1) / No(0) |
| **Q4. 평균 나이 도출 (피벗 활용)** | 동일한 피벗 테이블 내에 나이(`Age`) 데이터를 사용하여 **전체 평균 나이**가 정상적으로 분석되어 있습니까? | Yes(1) / No(0) |
| **Q5. 중위 나이 도출 (피벗 활용)** | 동일한 피벗 테이블 내에 나이(`Age`) 데이터를 사용하여 **전체 중위 나이(중앙값)**가 정상적으로 분석되어 있습니까? | Yes(1) / No(0) |
| **Q6. 생존 결과 교차 집계 (피벗 활용)** | 피벗 테이블 행(Row)에 성별이나 객실 등급을 넣고, 값(Values)에 생존 여부를 넣어 **다차원적인 교차 집계표**가 완성되어 있습니까? | Yes(1) / No(0) |

* 총점: 6점 만점 (각 문항 1점)
* **평가 시 주의사항**: 구글 시트 곳곳을 클릭하여, 동료가 단순 텍스트 캡처나 타이핑을 한 것이 아니라 실제로 마법(스마트 채우기)과 피벗 테이블 기능을 능숙하게 다뤘는지 점검해 보세요.
