# 6주차 실습 동료평가 기준표 (Peer Evaluation Rubric)

동료가 eCampus에 제출한 [실습 결과물 캡처 이미지]를 보고 아래 3가지 항목에 대해 객관적으로 채점(Yes / No)해 주시기 바랍니다.

| 평가 항목 | 채점 기준 | 배점 |
| :--- | :--- | :--- |
| **Q1. 데이터 검색 및 구조화 여부** | 제출된 화면에 파편화된 데이터가 구글 스프레드시트의 행과 열에 맞게 **표 형태로 깔끔하게 복사/붙여넣기** 되어 있습니까? (단순한 텍스트 뭉치가 아님) | Yes(1) / No(0) |
| **Q2. 틀 고정 적용 여부** | 캡처 화면 상단 1행(또는 특정 행)을 보면 행 번호선 아래에 **틀 고정(회색 굵은 선)**이 명확하게 적용되어 제목 줄이 고정되어 있습니까? | Yes(1) / No(0) |
| **Q3. 정렬 및 조건부 서식(색상 스케일) 적용 여부** | 특정 숫자 열을 기준으로 데이터가 오름차순 또는 내림차순으로 **정렬**되어 있으며, 숫자 크기에 따라 셀 배경에 **조건부 서식(색상 스케일/그라데이션)**이 적용되어 시각적으로 구분이 갑니까? | Yes(1) / No(0) |

* 총점: 3점 만점 (각 문항 1점)
* **평가 시 주의사항**: 문법이나 내용의 정답 여부를 채점하는 것이 아닙니다. 동료가 "검색한 데이터를 시트로 옮기고, 틀 고정과 색상 스케일을 통해 시각적으로 정리했는가"라는 도구 조작의 수행 여부만 확인합니다.
