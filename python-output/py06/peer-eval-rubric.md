# 6주차 실습 동료평가 기준표 (Peer Evaluation Rubric)

동료가 eCampus에 제출한 **[구글 시트 공유 링크]**에 접속하여 아래 5가지 항목에 대해 객관적으로 채점(Yes / No)해 주시기 바랍니다. (링크가 제대로 열리지 않으면 0점을 부여할 수 있습니다.)

| 평가 항목 | 채점 기준 | 배점 |
| :--- | :--- | :--- |
| **Q1. 데이터 정상 가져오기** | 제출된 구글 시트에 타이타닉 탑승객 데이터(csv)가 온전히 불러와져 하나의 시트에 정리되어 있습니까? | Yes(1) / No(0) |
| **Q2. 호칭 추출 분리** | `Name` 컬럼 옆에 **'호칭' 전용 열이 새롭게 추가되어 'Mr., Miss., Mrs.' 등의 데이터가 800여 개 모두 채워져 분리**되어 있습니까? | Yes(1) / No(0) |
| **Q3. 총 승선객 수 세기 (피벗 활용)** | 분석용 피벗 테이블이 별도로 생성되어 있으며, 이를 통해 타이타닉의 **'총 탑승객 수(개수)'** 값이 화면에 도출되어 있습니까? | Yes(1) / No(0) |
| **Q4. 평균 및 중위 나이 도출 (피벗 활용)** | 동일한 피벗 테이블 내에서 나이(`Age`) 데이터를 두 번 사용하여 **전체 평균 나이와 전체 중위 나이(중앙값)**가 정상적으로 분석되어 있습니까? | Yes(1) / No(0) |
| **Q5. 성별/Pclass별 교차 집계 (피벗 활용)** | 동일한 피벗 테이블 행(Row)에 성별이나 객실 등급을 넣고, 값(Values)에 생존 여부를 넣어 **다차원적인 생존 결과 교차 집계표**가 완성되어 있습니까? | Yes(1) / No(0) |

* 총점: 5점 만점 (각 문항 1점)
* **평가 시 주의사항**: 동료가 단순 텍스트 타이핑이나 캡처가 아닌, **'읽기 전용 구글 시트 링크'**를 올바르게 제출했는지 확인해야 합니다. 동료가 마법(스마트 채우기)이나 피벗 테이블 설정을 올바르게 다뤘는지 시트 곳곳을 클릭해 점검해 보세요.
