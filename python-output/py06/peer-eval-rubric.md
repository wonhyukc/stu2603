# 6주차 실습 동료평가 기준표 (Peer Evaluation Rubric)

동료가 eCampus에 제출한 **[구글 시트 공유 링크]**에 접속하여 아래 3가지 항목에 대해 객관적으로 채점(Yes / No)해 주시기 바랍니다. (링크가 제대로 열리지 않으면 0점을 부여할 수 있습니다.)

| 평가 항목 | 채점 기준 | 배점 |
| :--- | :--- | :--- |
| **Q1. 데이터 준비 및 호칭 추출** | 제출된 구글 시트에 타이타닉 탑승객 데이터가 온전히 불러와져 있으며, `Name` 컬럼 옆에 **'호칭' 전용 열이 새롭게 추가되어 'Mr., Miss., Mrs.' 등의 호칭 데이터가 끝까지 전부 분리(추출)되어** 있습니까? | Yes(1) / No(0) |
| **Q2. 통계 함수 사용 기초** | 빈 셀 어딘가에 `=COUNTA()`, `=AVERAGE()`, `=MEDIAN()` 등의 통계 함수 등을 성공적으로 사용하여 **[타이타닉 총 탑승객 수 / 전체 평균 나이 / 전체 중위 나이]** 세 가지 숫자를 모두 올바르게 계산하여 띄워 두었습니까? | Yes(1) / No(0) |
| **Q3. 복합 집계 (성별 및 Pclass별 생존자)** | `=COUNTIFS()` 수식이나 **[피벗 테이블]** 기능을 성공적으로 활용하여, 남성/여성에 따른 **성별 생존자 수**와 객실 등급(**Pclass**)에 따른 생존자 수가 교차 집계된 결과가 화면에 뚜렷하게 도출되어 있습니까? | Yes(1) / No(0) |

* 총점: 3점 만점 (각 문항 1점)
* **평가 시 주의사항**: 동료가 캡처 이미지가 아닌, 원본 수식과 피벗 테이블을 직접 확인할 수 있는 **'읽기 전용 구글 시트 링크'**를 제출했는지 확인해야 합니다. 동료가 마법(스마트 자동 채우기)이나 적절한 함수를 썼는지 결과 셀을 클릭해 수식 입력줄을 가볍게 점검해 보세요.
