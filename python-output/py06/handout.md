## 2. 실습 섹션 (Kaggle 타이타닉 데이터 분석 미션)

오늘 배운 구글링 스킬과 구글 스프레드시트 기초 함수 역량을 결합하는 실습 미션입니다. 미션을 완수하고 결과가 명확히 담긴 구글 시트 화면 전체를 스크린샷 캡처하여 **eCampus 실습 과제란에 업로드** 하세요.

**[Step-by-Step 실습 미션 워크플로우]**

1. **데이터 가져오기 (Kaggle)**: 구글에서 `Kaggle Titanic Dataset`을 검색하여, 캐글 사이트에 가입 후 직접 `train.csv` 파일을 다운로드합니다. 빈 구글 스프레드시트에 [파일] - [가져오기]를 통해 데이터를 깔끔하게 불러옵니다.
   *(⚠️ 직접 다운로드가 도저히 안 될 경우, 강사가 제공한 [타이타닉 트레인 셋 백업 링크](https://docs.google.com/spreadsheets/d/1___xpsm5ie5XwWPIL8D8OaMQkQQuBazyodXdQRruQiI/edit?usp=sharing)에 접속해 [파일]-[사본 만들기]로 가져와 시작하세요.)*
2. **이름에서 '호칭' 분리하기**: `Name` 컬럼(열) 바로 오른쪽에 새로운 열을 하나 삽입하고, 가장 위 제목에 `호칭`이라고 적습니다. 승객 이름(예: `Braund, Mr. Owen Harris`) 부분에서 'Mr.', 'Mrs.', 'Miss.' 등의 호칭 부분만 떼어내어 새 열을 전부 채우세요. (팁: 구글 시트의 [데이터] - [스마트 자동 채우기] 또는 텍스트 함수, [텍스트를 열로 분할] 등을 구글링하여 스스로 해결책을 찾습니다.)
3. **승선객 전원의 총숫자 세기**: 데이터 옆 빈 공간에 `=COUNTA()` 등의 함수를 사용해 데이터에 기록된 탑승객 전체 숫자를 세어 화면에 띄우세요.
4. **평균 및 중위 나이 도출**: `Age` 컬럼을 활용해 타이타닉 탑승객 전체의 **평균 나이(`=AVERAGE`)**와 일렬로 세웠을 때 정중앙에 있는 **중위 나이(`=MEDIAN`)**를 각각 빈 셀에 수식으로 구합니다.
5. **성별 생존자 수 도출**: 데이터의 `Sex`와 `Survived` 컬럼을 활용해 생존한 남성과 여성의 숫자를 각각 정확히 구하세요. (팁: 조건에 맞는 항목만 세는 `=COUNTIFS()` 함수를 검색해보거나, [피벗 테이블] 기능을 활용하면 쉽게 풀립니다.)
6. **캡처 후 제출**: 원본 데이터 표, 직접 만든 '호칭' 열, 그리고 여러분이 함수로 도출해낸 **'총 승선객 수 / 평균 나이 / 중위 나이 / 성별 생존자 수'**가 화면 한쪽에 빠짐없이 보이도록 캡처하여 eCampus에 제출합니다!
