## 2. 실습 섹션 (Google Sheets 구조화 미션)

오늘 배운 구글링 스킬과 구글 스프레드시트 기초를 결합하는 실습 미션입니다. 성공적으로 수행한 후, 완성된 구글 시트 화면 전체를 스크린샷 캡처하여 **eCampus 실습 과제란에 업로드** 하세요. (동료들에 의해 3가지 항목으로 상호평가 됩니다.)

**[Step-by-Step 실습 미션 워크플로우]**

1. **팩트 사냥**: 구글에서 자신이 관심 있는 주제(예: 국가별 인구, OECD 경제 지표, 특정 브랜드 매출 등)를 검색하되, 반드시 **`-` 연산자, `site:` 또는 `filetype:pdf` 중 하나 이상을 사용하여** 신뢰할 수 있는 공식 표(Table)나 수치 데이터를 찾아냅니다.
2. **복사 및 값만 붙여넣기**: 찾아낸 데이터 표를 드래그하여 복사한 뒤, 빈 구글 스프레드시트를 생성하고 1행 1열(A1)에 단축키(`Ctrl+Shift+V`)를 명심하여 '값만 붙여넣기' 합니다.
3. **틀 고정 (Freeze)**: 복사해 온 데이터의 가장 첫 번째 줄(분류, 연도, 국가 등 제목이 적힌 1행)을 스크롤을 내려도 항상 보이도록 구글 시트의 [보기] - [틀 고정] 기능을 적용합니다.
4. **오름차순/내림차순 정렬 (Sort)**: 본인이 중요하다고 생각하는 숫자 데이터의 열(알파벳렬)을 기준으로 시트 전체를 '내림차순(가장 큰 숫자부터)' 또는 '오름차순(작은 숫자부터)'으로 데이터 세트를 촤르륵 재정렬해 봅니다.
5. **조건부 서식 (색상 스케일 적용)**: 정렬된 해당 숫자 셀들을 마우스로 길게 드래그한 뒤, 메뉴의 [서식] - [조건부 서식] - [색상 스케일] 탭에 들어가 숫자 크기에 따라 붉은색~푸른색 등으로 시각적인 그라데이션이 퍼지도록 만듭니다.
6. **캡처 후 제출**: 제목 틀 고정선과 화려한 색상 스케일이 모두 나오도록 전체 화면을 캡처(PrtSc 또는 화면 캡처 도구)하여 eCampus에 제출합니다!
