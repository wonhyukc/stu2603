## 📌 이번 주 핵심 학습 키워드 (Weekly Overview)
이번 주에는 방대한 정보 쓰레기 속에서 진짜 팩트만 찾아내어 구글 시트로 깔끔하게 정리하고 기능적으로 가공하는 실무 훈련을 진행했습니다. 아래의 5가지 핵심 기술을 익혔습니다:
1. **스팸 및 낚시성 정보 필터링 (고급 구글 검색)**: `" "`, `-`, `site:`, `filetype:pdf` 등의 검색 연산자를 활용하여 상업적 광고나 불필요한 기사를 배제하고, 논문 등 신뢰할 수 있는 공식 리포트만 전천후로 스캐닝하기
2. **원시 데이터 업로드 및 표 구조화**: 산발적인 수치 데이터나 방대한 `.csv` 파일들을 무지성 텍스트 복붙이 아닌, '가져오기'와 '구분자 분할'을 통해 구글 시트의 완벽한 표 형태로 구조화하기
3. **가독성 극대화 및 기초 시각화 기술**: 열과 행이 길어져도 제목 줄을 유지하는 **'틀 고정'**, 데이터를 순위별로 정렬하는 **'오름차순/내림차순 정렬'**, 수치의 크기에 따라 열화상 카메라처럼 색상을 칠해주는 **'셀 조건부 서식(색상 스케일)'**을 적용하여 원시 데이터를 한눈에 파악하기 
4. **스마트 전처리 (데이터 분리 마법)**: 구글 시트의 자동화 기능인 '스마트 자동 채우기'를 통해 수천 개의 이름 텍스트 뭉치에서 원하는 패턴(호칭)만 순식간에 추출 및 분리해내기
5. **피벗 테이블 다차원 분석**: 복잡한 통계 함수 도움 없이 오직 피벗 테이블 조작만으로 대량의 데이터를 요약해 총 탑승객 수, 평균 나이, 중앙값을 구하고 '성별 및 객실 등급별 생존율' 같은 다차원 교차 집계를 단숨에 완성하기

### 💡 구글 시트로 외부 데이터(CSV, 엑셀) 가져오기 3가지 완벽 매뉴얼
실무에서는 데이터를 구글 시트에 넣기 위해 단순히 무지성 복사/붙여넣기만 쓰지 않습니다. 상황에 따라 다음 세 가지 방식을 유연하게 선택하세요.

1. **단순 복사 후 '값만 붙여넣기' (`Ctrl+Shift+V`)**
   - **용도**: 위키백과 등 일반 웹사이트에 있는 가벼운 표를 복사해서 빠르게 가져올 때 씁니다.
   - **특징**: 깨진 폰트나 눈 아픈 배경색 등 원본의 지저분한 서식을 싹 모두 버리고, 오직 순수한 '글자와 숫자 데이터'만 담백하게 불러올 수 있습니다.
2. **[파일 가져오기] 메뉴로 CSV 직접 업로드 및 변환 (Kaggle 등 공공데이터 필수)**
   - **용도**: 쉼표로만 콤마콤마 이어진 방대한 `.csv` 텍스트 데이터를 엑셀 칸에 아름답게 분할해 넣을 때 씁니다.
   - **명령 체계**: 빈 시트 상단 메뉴에서 `[파일] -> [가져오기] -> [업로드 탭]`을 클릭해 다운받은 파일을 선택합니다.
   - **변환 옵션(매우 중요)**: 팝업창에서 가져오기 위치를 **'현재 시트 바꾸기'**로, 구분자 유형을 **'자동 감지(또는 쉼표)'**로 확인 후 [데이터 가져오기] 버튼을 누르세요. 엉망이던 텍스트가 마법처럼 표 형태로 자동 분할됩니다.
3. **기존 엑셀 파일(.xlsx) 드라이브 업로드 후 호환 버튼 누르기**
   - **용도**: 이미 예쁘게 만들어둔 내 컴퓨터의 엑셀 파일을 팀원들과 공유하며 실시간 동시 작업을 하고 싶을 때.
   - **방법**: 구글 드라이브(내 드라이브) 바탕 화면에 데스크톱 엑셀 파일을 쭉 드래그해서 놓아 업로드합니다. 이후 해당 파일을 더블클릭해서 열면 화면 상단 중앙 쪽에 나타나는 **[Google Sheets로 열기 (또는 상단 초록선)]** 버튼을 누르면 끝납니다.

## 2. 실습 섹션 (Kaggle 타이타닉 데이터 분석 미션)

오늘 배운 구글링 스킬과 구글 스프레드시트 기초 함수 역량을 결합하는 실습 미션입니다. 미션을 완수하고 결과가 명확히 담긴 구글 시트 화면 전체를 스크린샷 캡처하여 **eCampus 실습 과제란에 업로드** 하세요.

**[Step-by-Step 실습 미션 워크플로우]**

1. **데이터 가져오기 (Kaggle) [배점: 1점]**: 구글에서 `Kaggle Titanic Dataset -"광고"` 처럼 제외 연산자(`-`)나 쌍따옴표(`""`)를 활용한 고급 검색으로 캐글 사이트에 가입 후 `train.csv` 파일을 다운로드합니다. 빈 구글 스프레드시트에 [파일]-[가져오기]를 통해 데이터를 표 형태로 깔끔하게 완전히 불러옵니다. *(⚠️ 직접 다운로드가 도저히 안 될 경우 강사가 제공한 [데이터 백업본 링크](https://docs.google.com/spreadsheets/d/1___xpsm5ie5XwWPIL8D8OaMQkQQuBazyodXdQRruQiI/edit?usp=sharing)를 복사해 쓰세요.)*
2. **첫 행 틀 고정 [배점: 1점]**: 방대한 데이터를 마우스로 스크롤할 때 제목 줄이 사라지지 않고 항상 상단에 고정되어 보이도록 보기 메뉴에서 **'틀 고정(1행)'**을 설정하세요.
3. **오름차순/내림차순 정렬 [배점: 1점]**: 운임(`Fare`)이나 탑승객 나이(`Age`) 열 전체를 선택하여 오름차순 또는 내림차순으로 전체 데이터를 한 번 정렬해 보는 기초 시각화 작업을 수행하세요.
4. **조건부 서식(색상 스케일) 적용 [배점: 1점]**: 생존 여부(`Survived`)나 객실 등급(`Pclass`) 등 숫자로 이루어진 열을 하나 선택한 후 상단의 **[형식] - [조건부 서식] - [색상 스케일]** 탭을 클릭하여 수치의 크고 작음이 열화상 카메라처럼 한눈에 보이고 구분되게 만드세요.
5. **이름에서 '호칭' 똑똑하게 분리하기 [배점: 1점]**: `Name` 컬럼(열) 오른쪽 공간에 새로운 열을 하나 삽입하고 제목에 `호칭`이라고 적습니다. 승객 이름 부분에서 'Mr.', 'Mrs.', 'Miss.' 등의 호칭만 추출해 직접 세 개 이상 연속으로 빈칸을 채워보세요. 어느 순간 구글 시트가 패턴을 파악하여 띄우는 **스마트 자동 채우기(회색 미리보기)**를 작동시켜 800여 명의 데이터를 한 번에 분리해 완성합니다.
6. **승선객 총숫자 세기 (피벗 활용) [배점: 1점]**: 상단 메뉴의 **[삽입] - [피벗 테이블]**을 클릭해 **'새 시트'**에 전용 분석 공간을 만듭니다. 값(Values) 영역에 아무 데이터를 넣어 전체 탑승객 숫자(요약 기준: 개수)를 도출하세요.
7. **평균 나이 도출 (피벗 활용) [배점: 1점]**: 생성한 피벗 테이블의 값 영역에 `Age` 컬럼을 넣고 요약 기준을 **평균(AVERAGE)**으로 변경해 전체 탑승자의 평균 나이를 구하세요.
8. **중위 나이 도출 (피벗 활용) [배점: 1점]**: 구글 시트가 팩트를 기반으로 이상치(Outlier)에 영향을 덜 받는 값을 보여줄 수 있도록, 다시 한번 `Age` 컬럼을 값 영역에 추가하고 요약 기준을 **중앙값(MEDIAN)**으로 변경해 평균과 나란히 보이게 만드세요.
9. **성별/Pclass별 생존자 교차 집계 (피벗 활용) [배점: 1점]**: 피벗 테이블의 행(Row) 영역에 `Sex`(성별)와 `Pclass`(객실 등급)를 넣고, 값 영역에 생존 여부(`Survived`) 항목을 넣어 클릭 몇 번만으로 성별, 등급별 생존 격차가 명확히 보이는 교차 분석표를 완성해 내세요.
10. **최종 과제 제출 (필수 Fail-Fast)**: 위 과정에서 획득한 실무 팁이 모두 담긴 **타이타닉 구글 시트 공유 링크(보기 권한 필수)**를 eCampus에 제출합니다. (링크가 미작동하면 즉시 0점 처리됩니다.)
