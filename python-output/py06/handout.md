### 💡 구글 시트로 외부 데이터(CSV, 엑셀) 가져오기 3가지 완벽 매뉴얼
실무에서는 데이터를 구글 시트에 넣기 위해 단순히 무지성 복사/붙여넣기만 쓰지 않습니다. 상황에 따라 다음 세 가지 방식을 유연하게 선택하세요.

1. **단순 복사 후 '값만 붙여넣기' (`Ctrl+Shift+V`)**
   - **용도**: 위키백과 등 일반 웹사이트에 있는 가벼운 표를 복사해서 빠르게 가져올 때 씁니다.
   - **특징**: 깨진 폰트나 눈 아픈 배경색 등 원본의 지저분한 서식을 싹 모두 버리고, 오직 순수한 '글자와 숫자 데이터'만 담백하게 불러올 수 있습니다.
2. **[파일 가져오기] 메뉴로 CSV 직접 업로드 및 변환 (Kaggle 등 공공데이터 필수)**
   - **용도**: 쉼표로만 콤마콤마 이어진 방대한 `.csv` 텍스트 데이터를 엑셀 칸에 아름답게 분할해 넣을 때 씁니다.
   - **명령 체계**: 빈 시트 상단 메뉴에서 `[파일] -> [가져오기] -> [업로드 탭]`을 클릭해 다운받은 파일을 선택합니다.
   - **변환 옵션(매우 중요)**: 팝업창에서 가져오기 위치를 **'현재 시트 바꾸기'**로, 구분자 유형을 **'자동 감지(또는 쉼표)'**로 확인 후 [데이터 가져오기] 버튼을 누르세요. 엉망이던 텍스트가 마법처럼 표 형태로 자동 분할됩니다.
3. **기존 엑셀 파일(.xlsx) 드라이브 업로드 후 호환 버튼 누르기**
   - **용도**: 이미 예쁘게 만들어둔 내 컴퓨터의 엑셀 파일을 팀원들과 공유하며 실시간 동시 작업을 하고 싶을 때.
   - **방법**: 구글 드라이브(내 드라이브) 바탕 화면에 데스크톱 엑셀 파일을 쭉 드래그해서 놓아 업로드합니다. 이후 해당 파일을 더블클릭해서 열면 화면 상단 중앙 쪽에 나타나는 **[Google Sheets로 열기 (또는 상단 초록선)]** 버튼을 누르면 끝납니다.

## 2. 실습 섹션 (Kaggle 타이타닉 데이터 분석 미션)

오늘 배운 구글링 스킬과 구글 스프레드시트 기초 함수 역량을 결합하는 실습 미션입니다. 미션을 완수하고 결과가 명확히 담긴 구글 시트 화면 전체를 스크린샷 캡처하여 **eCampus 실습 과제란에 업로드** 하세요.

**[Step-by-Step 실습 미션 워크플로우]**

1. **데이터 가져오기 (Kaggle) [배점: 1점]**: 구글에서 `Kaggle Titanic Dataset`을 검색하여, 캐글 사이트에 가입 후 직접 `train.csv` 파일을 다운로드합니다. 빈 구글 스프레드시트에 [파일] - [가져오기]를 통해 데이터를 깔끔하게 불러옵니다.
   *(⚠️ 직접 다운로드가 도저히 안 될 경우, 강사가 제공한 [타이타닉 트레인 셋 백업 링크](https://docs.google.com/spreadsheets/d/1___xpsm5ie5XwWPIL8D8OaMQkQQuBazyodXdQRruQiI/edit?usp=sharing)에 접속해 [파일]-[사본 만들기]로 가져와 시작하세요. 하지만 나중에 기말 실기 시험은 이런 파일 다운로드 과정을 스스로 하도록 요구합니다. 반드시 지금 직접 해보기를 강력히 권합니다.)*
2. **이름에서 '호칭' 똑똑하게 분리하기 [배점: 1점]**: `Name` 컬럼(열) 바로 오른쪽에 새로운 열을 하나 삽입하고, 가장 위 제목에 `호칭`이라고 적습니다. 승객 이름 부분에서 'Mr.', 'Mrs.', 'Miss.' 등의 호칭만 떼어내어 직접 세 개 이상 직접 타이핑해서 빈칸을 채워 내려가 보세요. 어느 순간 구글 시트가 여러분의 의도를 파악하고 **마법처럼 회색 글씨로 미리보기(스마트 자동 채우기)**를 띄웁니다. 이때 엔터를 치면 800여 명의 데이터가 한 번에 완성됩니다. 이 마법을 경험하지 못했다면 백스페이스를 누르고 다시 시도해 꼭 익히세요. (수동으로 800개를 치지 않으려면 필수입니다.)
3. **승선객 전원의 총숫자 세기 (피벗 테이블 활용) [배점: 1점]**: 상단 메뉴의 **[삽입] - [피벗 테이블(Pivot Table)]**을 클릭한 뒤, 팝업창에서 **'새 시트'**를 명확히 선택하여 전용 분석 공간을 만듭니다. 이후 우측 메뉴의 값(Values) 영역에 아무 데이터나 넣어 전체 탑승객 숫자(요약 기준: 개수)를 직관적으로 도출해 보세요.
4. **평균 및 중위 나이 도출 (피벗 테이블 활용) [배점: 1점]**: 생성한 피벗 테이블의 값 영역에 이번엔 `Age` 컬럼을 2번 추가합니다. 그리고 각 항목의 우측 하단 클릭 후 요약 기준을 **평균(AVERAGE)**과 **중앙값(MEDIAN)**으로 각각 변경하여 전체 나이 통계를 완성합니다.
5. **성별 및 Pclass별 생존자 교차 집계 (피벗 테이블 활용) [배점: 1점]**: 피벗 테이블의 행(Row) 영역에 `Sex`(성별)와 `Pclass`(객실 등급)를 넣고, 값 영역에 생존 여부(`Survived`) 항목을 넣어 클릭 몇 번만으로 성별과 등급별 생존 격차가 명확히 보이는 교차 분석표를 최종 완성하세요.
6. **최종 2개의 과제 링크 제출 (필수 조건)**: 위에서 준비해둔 **1) AI 복습 교안(Gemini 등) 대화창 공유 링크**와 방금 완성한 **2) 나만의 타이타닉 구글 시트 공유 링크(보기 권한 필수)** 두 가지를 eCampus 실습 과제란에 모두 제출합니다. (링크가 권한 문제 등으로 열리지 않으면 해당 항목은 0점 처리됩니다.)
