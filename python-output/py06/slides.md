# 6주차: 정보 사냥꾼과 팩트 체크
## 고급 구글 검색과 데이터 구조화
**컴퓨팅적 사고와 코딩 (K-Track)**

---

# 📌 이번 주 핵심 목표

- **진짜 팩트 찾아내기**: 정보 쓰레기 속 신뢰도 높은 자료 발굴
- **구글 시트 구조화**: 파편화된 데이터와 CSV를 깔끔한 표로 조립
- **시각적 가공**: 틀 고정, 정렬, 조건부 서식으로 가독성 극대화
- **피벗 테이블 다차원 분석**: 복잡한 데이터 요약과 교차 집계

---

# 🛡️ 스팸을 거르는 고급 구글링 (1)
### 단어 제외 연산자: `-` (마이너스)
- 검색 결과에서 불필요한 단어를 배제
- 광고나 낚시성 기사를 거를 때 유용
- **예시**: `커피 효능 -광고 -쇼핑몰`

---

# 🛡️ 스팸을 거르는 고급 구글링 (2)
### 출처 지정 연산자: `site:`
- 특정 사이트, 도메인에서만 검색
- 정부, 대학 등 공신력 있는 자료만 추출
- **예시**: `기후 변화 site:go.kr` / `site:ac.kr`

---

# 🛡️ 스팸을 거르는 고급 구글링 (3)
### 파일 지정 연산자: `filetype:`
- 특정 확장자 형태의 파일만 검색
- 논문이나 공식 발표 형태인 PDF 탐색에 절대적
- **예시**: `인공지능 트렌드 filetype:pdf` 
  (**`"정확한 문구"`**와 조합하면 효과 만점!)

---

# 💡 외부 데이터 가져오기 3가지 기술

1. **단순 웹 표 복사**: `Ctrl+Shift+V` (값만 붙여넣기)
   - 웹사이트의 지저분한 서식(색상, 폰트)을 제거하고 순수 데이터만 추출
2. **CSV 파일 업로드**: 방대한 데이터 변환
   - **[파일] - [가져오기] - [업로드]**
   - 쉼표 데이터가 마법처럼 분할됨! (구분자: '자동 감지' ➡️ '현재 시트 바꾸기')
3. **엑셀(.xlsx) 파일 호환**: 드라이브 업로드 후 동시 협업
   - 구글 드라이브에 올린 뒤 **[Google Sheets로 열기]** 버튼 클릭

---

# 🏗️ 데이터를 한 눈에: 뼈대 잡기

### `틀 고정` 활용하기
- 데이터의 제목(Header)이 되는 1행을 고정
- **[보기] - [틀 고정] - [행 1개]**
- 방대한 데이터를 스크롤해도, 어떤 항목인지 길을 잃지 않음

---

# 🔍 시각적 정렬 1단계: 정렬(Sort)

- 무작위 데이터를 특정 기준에 맞게 재배열
- 알파벳 열 클릭 ➡️ **시트 정렬**
  - `오름차순 (A-Z)`: 작은 것부터 큰 것으로
  - `내림차순 (Z-A)`: 큰 것부터 작은 것으로
- 순위표 구조를 수초 만에 완성

---

# 🎨 시각적 정렬 2단계: 조건부 서식

### 색으로 말하는 데이터
- 범위 선택 후 **[서식] - [조건부 서식] - [색상 스케일]**
- 수치의 높고 낮음에 따라 열화상 카메라처럼 그라데이션 적용
- 문법적 코딩 한 줄 없이, 훌륭한 **데이터 시각화** 구현!

---

# ✨ 스마트 전처리: 데이터 분리 마법

### **스마트 자동 채우기**
- 수많은 텍스트 뭉치에서 원하는 패턴만 빠르게 쏙 골라내기
- 예: `Name` 데이터에서 'Mr.', 'Miss.' 같은 **호칭만 분리**하고 싶을 때
- 옆 칸에 직접 3번 정도 반복해서 적으면, 시트가 패턴을 파악하여 800여 명의 빈칸을 한 번에 자동으로 채워줌 (회색 미리보기 시 엔터!)

---

# 📊 피벗 테이블 다차원 분석 기초

### 클릭 몇 번으로 끝내는 데이터 요약
- **[삽입] - [피벗 테이블]** ➡️ '새 시트'
- 거대한 데이터의 총합, 평균, 중앙값을 복잡한 수학 함수 없이 단숨에 도출
- **사용법**:
  - `값(Values)` 영역에 데이터 추가 후 기준 변경(개수 ➡️ 평균 ➡️ 중앙값)
  - 이상치(Outlier)에 강한 팩트: **평균(AVERAGE)** vs **중앙값(MEDIAN)** 비교

---

# 🧩 교차 집계표 완성하기 (피벗 심화)

### 다차원 데이터 비교 분석
- 행(Row)과 값(Values)을 교차하여 새로운 인사이트 발견
- **Kaggle 타이타닉 데이터 예시**:
  - `행`: Sex(성별), Pclass(객실 등급)
  - `값`: Survived(생존 여부)
- ➡️ 성별과 객실 등급에 따른 생존 격차를 보여주는 입체적 교차 분석표 완성!

---

# 📝 Kaggle 타이타닉 실습 미션

1. `Kaggle Titanic Dataset -광고` 검색 후 `train.csv` 구글 시트 **가져오기**
2. 데이터 **틀 고정(1행)** 및 운임/나이 기준 **정렬**
3. 눈에 띄는 **조건부 서식(색상 스케일)** 적용
4. **스마트 자동 채우기**로 `이름` 열에서 호칭(Mr. 등) 분리해 새 열 만들기
5. **피벗 테이블** 생성: 총 탑승객 수 / 평균 나이 / **중앙값** 도출
6. **피벗 테이블** 심화: 성별/객실 등급별 생존율 **교차 집계표** 구성
7. 완성 화면 캡처하여 **eCampus 제출** + **AI 복습 교안 링크** 제출
