# 중간고사 상호평가(Workshop) 모듈용 문제 및 루브릭

강사님께서 15개의 문항을 일일이 문제은행에 등록하는 번거로움을 피하고, **무들 상호평가(Workshop) 모듈**에서 한 번에 복사/붙여넣어 운영하실 수 있도록 형태를 완전히 개편했습니다. 
무들의 편집기(에디터)에 바로 붙여넣었을 때 **줄바꿈(엔터)이 무시되는 현상을 방지하기 위해 HTML 태그를 적용**했습니다. 

편집기의 **`</>` (HTML 소스 편집)** 버튼을 누르고 아래 코드를 복사해서 붙여넣으시면 서식이 완벽하게 유지됩니다.

---

## 1. 제출 지시문 (Instructions for submission)
> **사용 방법**: 상호평가 설정의 '제출 설정' -> '제출 지시문' 에디터에서 `</>` (HTML 소스 편집) 창을 열고 아래 코드를 그대로 붙여넣으세요.

```html
<h3>[파이썬 프로그래밍 중간고사 타임어택 미션]</h3>
<p>본 중간고사는 여러분의 '디지털 생존력'과 'AI 도구 활용 속도'를 평가합니다. 아래 15개의 문항을 순서대로 읽고, 제한 시간 내에 모든 정답을 작성하여 제출해 주시기 바랍니다. (AI 도구, 구글 검색, 번역기 등 모든 도구 활용 가능)</p>
<hr>

<p><b>📌 문항 1. [프롬프트 최적화] 치명적인 에러 질문법 (객관식)</b><br>
<b>상황:</b> 파이썬 코드를 실행하다가 다음 에러를 마주했습니다.<br>
<code>TypeError: unsupported operand type(s) for +: 'int' and 'str'</code><br>
<b>문제:</b> 이 에러를 해결하기 위해 AI(Gemini/ChatGPT)에게 던질 가장 효과적이고 빠른 프롬프트는 무엇입니까?<br>
① 파이썬 에러 났어. 고쳐줘.<br>
② "TypeError: unsupported operand type(s) for +: 'int' and 'str'" 에러의 원인이 뭐야?<br>
③ 다음 코드 블록을 실행할 때 "TypeError:..." 에러가 발생했어. 숫자와 문자를 더하려고 한 것 같은데 수정 코드를 제시해줘. [코드 첨부]<br>
④ 파이썬에서 int랑 str은 왜 더할 수 없어?</p>

<p><b>📌 문항 2. [파일 네이밍 규칙] 숨겨진 폭탄 찾기 (객관식)</b><br>
<b>상황:</b> 다음은 조별 과제 폴더에 취합된 파일 목록입니다.<br>
① <code>20240419_report_final.pdf</code><br>
② <code>2024-04-19_lecture_notes_v2.docx</code><br>
③ <code>240419_team_project-draft.pptx</code><br>
④ <code>20240419_Assignment_01.xlsx</code><br>
<b>문제:</b> 프로젝트 파일 명명 규칙(1. ISO 8601 기준 날짜 2. 소문자 스네이크 케이스 3. 띄어쓰기 금지)을 완벽하게 준수한 파일의 번호를 고르시오.</p>

<p><b>📌 문항 3. [검색 연산자 심화] 구글링 스피드런 (단답형)</b><br>
<b>상황:</b> 여러분은 정부의 '2024년도 "청년" 지원 정책'에 대한 엑셀(Excel) 원본 데이터를 찾으려고 합니다.<br>
<b>문제:</b> 네이버 블로그나 뉴스 기사를 제외하고, 오직 정부 기관 사이트(.go.kr)에서 제공하는 엑셀 파일만 정확하게 검색하기 위한 구글 검색어(연산자 포함)를 작성하세요.</p>

<p><b>📌 문항 4. [OS 확장자 이해] 위장된 악성코드 감별 (객관식)</b><br>
<b>상황:</b> 인터넷에서 실습 자료를 다운로드하여 압축을 풀었더니 다음과 같은 파일들이 나왔습니다. 윈도우에서 '알려진 파일 형식의 파일 확장명 숨기기'가 켜져 있는 상태입니다.<br>
① <code>homework_instructions.pdf</code> (PDF 아이콘)<br>
② <code>dataset_final.csv</code> (엑셀 아이콘)<br>
③ <code>lecture_video.mp4.exe</code> (동영상 플레이어 아이콘)<br>
④ <code>readme.txt</code> (메모장 아이콘)<br>
<b>문제:</b> 이 중 실제로 더블클릭했을 때 바이러스(실행 파일)일 가능성이 가장 높은 파일 번호를 고르시오.</p>

<p><b>📌 문항 5. [AI 데이터 전처리] 정규식을 활용한 데이터 추출 (서술형)</b><br>
<b>상황:</b> 이메일이 무작위로 섞여 있는 난잡한 텍스트 더미 파일이 주어졌습니다.<br>
<b>문제:</b> AI에게 어떤 프롬프트를 작성하여 이 텍스트에서 '정확한 이메일 주소 형식'을 가진 데이터만 추출하고 정렬했는지 본인의 파이프라인(사용한 프롬프트와 과정)을 간략히 서술하세요.</p>

<p><b>📌 문항 6. [AI 코드 리뷰] 의도된 버그 짚어내기 (단답형)</b><br>
<b>상황:</b> AI가 짜준 아래 파이썬 코드를 윈도우에서 실행하면 폴더 경로 문제로 에러가 발생합니다.<br>
<code>folder_path = "C:\Users\Student\Documents\Test"</code><br>
<b>문제:</b> 이 에러를 해결하기 위해 위 변수 선언 부분을 올바른 코드로 수정하여 작성하세요.</p>

<p><b>📌 문항 7. [경로 추적기] 상대/절대 경로 이해 (단답형)</b><br>
<b>상황:</b> 현재 작업 디렉토리가 <code>C:\Projects\Python\Week08</code> 입니다. 읽어야 할 데이터 파일은 <code>C:\Projects\Data\sales.csv</code> 에 있습니다.<br>
<b>문제:</b> 파이썬 코드에서 sales.csv를 읽어오기 위한 상대 경로(Relative Path)를 작성하세요.</p>

<p><b>📌 문항 8. [데이터 심폐소생술] 깨진 CSV 복구 타임어택 (서술형)</b><br>
<b>상황:</b> 인코딩이 깨지고 구분자가 세미콜론(;)으로 잘못된 CSV 파일이 주어졌을 때,<br>
<b>문제:</b> 이 파일을 정상적인 엑셀 표 형태로 복구하기 위해 본인이 어떤 방법(엑셀 기능, 메모장 인코딩, 파이썬 등)을 사용했는지 그 해결 과정을 서술하세요.</p>

<p><b>📌 문항 9. [외국어 에러 해독] 실시간 번역 릴레이 (객관식)</b><br>
<b>상황:</b> 프로그램 실행 중 다음과 같은 에러가 발생했습니다.<br>
<code>Ошибка: Недостаточно прав для записи в директорию /var/log/</code><br>
<b>문제:</b> 번역기나 AI를 활용해 위 메시지를 해독했을 때, 가장 적합한 원인을 고르시오.<br>
① 디스크 용량 부족<br>
② 관리자 권한 부족 (쓰기 권한 없음)<br>
③ 인터넷 연결 끊김<br>
④ 지원하지 않는 파일 형식</p>

<p><b>📌 문항 10. [블랙 아이스 함정] 시스템 탈주 (단답형)</b><br>
<b>상황:</b> 무들 퀴즈 창의 "정답 입력칸"이 의도적으로 HTML disabled 속성으로 잠겨 있어 타이핑이 불가능합니다.<br>
<b>문제:</b> 브라우저의 '개발자 도구(F12)'를 열고 입력칸을 해제하여 정답을 입력해야 합니다. 이 문항의 정답인 "HACKED"라는 단어를 제출란에 적으세요.</p>
```
