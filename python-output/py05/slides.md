## [S1 섹션 슬라이드: 문제 상황 직면]

### 실생활 문제 상황 1
- **상황**: 내일 아침 9시가 리포트 제출 마감인데, 실수로 리포트 파일의 내용을 지워버렸습니다! 마우스를 던지고 멘붕에 빠지시나요?
- **비교 화면**:
    - [X] 뒤통수 잡고 책상에 엎드린 인간
    - [O] 차분하게 키보드의 `Ctrl + Z` (실행 취소)를 두 번 눌러서 어제 만들어둔 상태로 복원하는 멋진 대학생

### 실생활 문제 상황 2
- **상황**: 알 수 없는 영어 경고창(Error)이 화면 한가운데에 떴습니다. 어떻게 행동합니까?
- **플로우 다이어그램**:
    - [일반인] "영어가 떴네?" ➔ 무서우니까 창의 [X] 버튼 클릭 ➔ 다음날 더 심각한 오류 발생 후 눈물바다
    - [생존자] 경고창 내용 드래그 복사 또는 폰 렌즈 사진 찍기 ➔ 한국어 번역기에 텍스트 붙여넣기 ➔ 구글에 원인 검색 후 "확인" 클릭

## [S2 섹션 슬라이드: 파일 관리 구조와 크롬 백배 활용 원리]

### 디렉토리(폴더) 저장의 나무 구조
- **개념 시각도표**: 
  - 🌳 바탕화면(Root)
    - 📂 내 리포트 보관함
      - 📄 1학기 정보 리포트
    - 📂 스마트폰 동기화 사진
- 폴더 안에 폴더를 나누어 효율적으로 정리하지 않으면 여러분의 컴퓨터 화면은 쓰레기통과 같아집니다!

### 기적의 시간 단축키 트리오
- **`Ctrl + C`** : 텍스트나 파일 복사 (카피)
- **`Ctrl + V`** : 지정된 위치에 붙여넣기 (페이스트)
- **`Ctrl + Z`** : 직전 행동 뒤로 되돌리기 (가장 소중한 인생의 세이브 포인트!)

### 크롬 브라우저 생존 스킬
- **시크릿 모드 (Ctrl+Shift+N)**: 공용 PC나 도서관 컴퓨터에서 내 구글 로그인 기록을 남기지 않고 은밀하게 검색하는 생존술.
- **언어 환경 설정**: 영어로만 적혀 나오는 외국 웹사이트를 즉시 한국판 메인 화면으로 전환하는 법.

## [S3 섹션 슬라이드: 라이브 데모 및 미션 안내]

### [강사 라이브 데모 데스크 플로우]
1. **문제 발생**: 강사가 크롬 브라우저 화면에 억지로 "404 Not Found" 라든지 "Permission Denied" 에러를 연출해 봅니다.
2. **진단**: 당황하지 않고 마우스 우클릭으로 번역하거나, 직접 그 텍스트를 파파고 번역기에 붙여 "권한 거부"임을 직독!
3. **치료**: 구글 검색창을 열어 "크롬 Permission Denied 해결 조치법" 검색. 블로그의 안내 글을 보고 클릭 3번으로 에러를 날려버리는 짜릿한 시연!

### 이번 주차 과제 워크플로우 안내
- **실습 미션**: 자신이 가지고 있는 컴퓨터(폰)에서 엉뚱한 에러 화면 혹은 뜻을 모르는 영어 설정창을 스스로 띄우고, 이 화면을 번역기로 번역하고 검색하여 기능이 무엇인지 알아낸 전 과정을 스크린샷으로 캡처하세요.
- **eCampus 동료평가 제출 대상**: 문제해결 전·후 스크린샷 이미지 뭉치 (PDF 등 1장 묶음 권장).
- **강사 이메일 제출 규정**: 결과물을 도출하며 마주친 번역기 결과물과 구글링 키워드를 "과정 서술형" 글로 짧게 요약하여 메일 전송! (`wonhyuk@stu.ac.kr`)
