# 과제 용어 및 제출 방식 변경 안내 / Notice on Assignment Terminology and Submission Methods

이번 주차부터 수강생 여러분의 혼란을 방지하고 명확한 안내를 제공하기 위해, 과제의 지칭 용어와 제출 방식을 다음과 같이 명확하게 분류하여 운영합니다.

Starting this week, to prevent any confusion and provide clear guidelines, we are strictly categorizing the assignment terminology and submission methods as follows.

## 1. 이메일 과제 / 1. Microassignment (Email Submission)

기존에 이메일로 제출하던 주차별 0.x 과제들은 앞으로 '이메일 과제(Microassignment)'로 통일하여 부릅니다. 제출 방식은 기존과 동일하게 여러분의 학번과 이메일을 사용해 정해진 기한 내에 제출하시면 됩니다.

The weekly 0.x assignments previously submitted via email will now be uniformly referred to as 'Microassignments'. The submission method remains the same; please submit them via email within the designated deadline using your student ID.

## 2. 실습 과제 / 2. Assignment (eCampus/Google Form Submission)

수업 내용이 포함된 메인 실습으로서, eCampus나 구글 폼을 통해 제출한 뒤 학생들 간의 상호평가(Peer Evaluation)를 진행하는 과목들은 이제 '실습 과제(Assignment)'로 부릅니다.

The main practical tasks covering the class content, which are submitted via eCampus or Google Forms and followed by student peer evaluations, will now be called 'Assignments'.

이 변경 사항은 이번 5주차 수업 자료(Handout)부터 즉시 적용되어 있습니다. 과제 수행 시 본인이 제출해야 하는 항목이 '이메일 과제'인지 '실습 과제'인지 명확히 확인해 주시기 바랍니다.

These changes have been applied immediately starting with the Week 5 course materials (Handout). When performing your tasks, please clearly check whether the item you need to submit is a 'Microassignment' or an 'Assignment'.
