## 1. 기적의 핵심 도구 요약표 (컨닝 페이퍼)

### [필수 도구 링크]
- **구글 번역(Google Translate)**: [translate.google.com](https://translate.google.com/) (복사한 텍스트를 즉시 번역)
- **구글 렌즈(Google Lens)**: 스마트폰 스토어에서 다운로드 (복사가 안 되는 에러창 사진을 찍어 화면 위에서 직접 번역)
- **구글 검색(Google Search)**: [google.com](https://google.com) (번역된 의미를 넣고 '해결법' 등 키워드 조합)

### [대학 생활 4년을 책임질 단축키 5종]

| 단축키 (Windows 기준) | 기능 | 생존 의미 |
|:---|:---|:---|
| **`Ctrl + Z`** | 실행 취소 (Undo) | 방금 지워버린 글, 이동시킨 파일을 직전 상태로 완벽히 되돌립니다. 인생의 세이브 포인트입니다. |
| **`Ctrl + C`** | 복사하기 (Copy) | 선택한 텍스트/파일을 클립보드에 복사합니다. 원본은 그대로 유지됩니다. |
| **`Ctrl + V`** | 붙여넣기 (Paste) | 복사한 것을 커서 위치에 즉시 붙여넣습니다. |
| **`Ctrl + A`** | 전체 선택 (Select All) | 문서 전체 또는 폴더 내 모든 파일을 한 번에 선택합니다. |
| **`Ctrl + S`** | 저장 (Save) | 작업 중인 파일을 저장합니다. 10분에 한 번씩 습관적으로 누르세요. |

Mac 사용자는 `Ctrl` 대신 `Command(Cmd)` 키를 사용합니다.

### [에러 직독직해 3단계 플로우]

| 단계 | 텍스트 복사 가능할 때 | 텍스트 복사 불가능할 때 |
|:---|:---|:---|
| 1단계 | 에러창 텍스트를 마우스 드래그 후 `Ctrl+C` | 구글 렌즈 앱 켜서 모니터 화면 비추기 |
| 2단계 | [translate.google.com](https://translate.google.com/)에 `Ctrl+V` 붙여넣기 | 렌즈 앱에서 '번역' 모드 선택 |
| 3단계 | 번역 결과 + "해결법" 키워드로 구글 검색 | 번역된 내용 + "해결법" 키워드로 구글 검색 |

### [크롬 브라우저 생존 스킬]

| 기능 | 방법 |
|:---|:---|
| 시크릿 모드 (공용 PC 필수) | `Ctrl + Shift + N` 또는 점 세 개 메뉴 → '새 시크릿 창' |
| 외국어 페이지 자동 번역 | 주소창 오른쪽 번역 아이콘 클릭 또는 설정 → 언어 → 자동 번역 활성화 |
| 새 탭 열기 | `Ctrl + T` |
| 페이지 새로고침 | `F5` |

## 2. 실습 (조작) 워크플로우 — eCampus 동료평가 제출 대상

오늘 실습의 목표는 **어떤 에러창이나 낯선 환경이 닥쳐도 당황하지 않고 번역과 검색을 통해 스스로 기능이나 해결책을 알아내는 것**입니다. 아래 순서도 그대로 미션을 수행하세요.

1. **상황 찾기**: 본인의 폰이나 컴퓨터 설정에 깊숙이 들어가 본 적 없는 알 수 없는 영문 메뉴를 발견하거나, 인터넷 접속 장애 등 실제 에러 팝업창을 스스로 하나 띄워봅니다.
2. **증거 수집**: 그 에러 화면(또는 낯선 메뉴 화면)을 캡처하거나 폰의 카메라로 선명하게 찍어 둡니다. (Before 화면 확보)
3. **번역 직독직해**:
    - 텍스트 복사가 가능하다면 → 마우스로 쭉 드래그해 `Ctrl+C` 복사 후, 구글 번역기에 `Ctrl+V`로 붙여넣기.
    - 텍스트 복사가 막혀 있다면 → 스마트폰 구글 렌즈(Google Lens) 앱을 켜고 화면을 비춘 뒤 '번역' 모드를 터치!
4. **구글링 투입**: 번역된 한국어 뜻을 파악한 뒤, 새 크롬 창을 열고 구글 검색창에 [그 에러 내용] + "해결법" 혹은 "무슨 기능"이라고 검색어 키워드를 입력합니다.
5. **문제 해결**: 검색 결과 상단에 뜬 가이드를 보며, 내 컴퓨터에서 세팅을 조작하여 해결해 봅니다.
6. **결과 수집**: 해결된 이후 바뀐 화면을 다시 캡처합니다. (After 화면 확보)

**eCampus 제출 방법**: Before 화면 + 번역/검색 화면 + After 화면을 순서대로 묶은 PDF 또는 이미지 파일 1개를 eCampus 과제 게시판에 업로드합니다. 동료 학생들이 확인하고 동료 평가를 진행합니다.
