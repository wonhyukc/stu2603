## 1. 개념 참고 영역 (Reference)

### (1) 변수 (Variables)란 무엇인가요?
변수(Variable)는 데이터를 담아두는 '상자' 또는 데이터에 이름표를 붙이는 것과 같습니다. 수학 시간에 배운 x, y와 비슷하지만, 파이썬에서는 이름표에 훨씬 더 자유로운 이름을 붙일 수 있습니다!
- **선언과 할당**: `age = 20` (age라는 이름의 상자에 20을 넣습니다)
- **복합 조작**: `age += 1`은 기존 나이에 1을 더한다는 뜻으로, `age = age + 1`과 완전히 동일합니다. 많이 쓰이니 눈에 익혀두세요!
- **이름 짓는 규칙 (snake_case)**: 단어와 단어 사이를 공백 대신 밑줄(`_`)로 연결하여 뱀처럼 보이게 합니다. (예: `student_name`, `my_age`). 파이썬에서는 **한글 변수명**(`학생_나이 = 20`)도 사용할 수 있습니다. 하지만 완전 자유로운 것은 아니니 주의해야 합니다.

### (2) 문자열 위치 찾기 (Indexing & Slicing)
'Hello' 라는 글자가 있을 때, 파이썬은 이 글자들의 순서를 **0부터 셉니다**. (이것은 프로그래머들의 세계적인 규칙입니다!)
`text = "Hello"`
- **인덱싱(콕 집어 꺼내기)**: `text[0]`은 첫 번째 글자인 `'H'`를 가져옵니다. `text[-1]`은 뒤에서 첫 번째, 즉 맨 끝 글자는 `'o'`를 가져옵니다. 외국인 유학생 여러분도 쉽게 이해할 수 있도록 맨 뒤부터는 -1, -2, -3으로 간다고 기억해 주세요.
- **슬라이싱(범위로 자르기)**: `[시작:끝:보폭]` 형태로 씁니다. 보폭은 생략할 수 있습니다. 주의할 점은 끝맺는 번호는 **포함하지 않는다**는 것입니다! `text[0:4]`라고 쓰면 0, 1, 2, 3번째 글자인 `'Hell'` 까지만 가져옵니다. 만약 `text[0:5:2]`처럼 콜론 두 개를 쓰면 0번부터 5번 직전까지 **2칸씩 건너뛰며** 가져오라는 뜻으로 `'Hlo'`가 됩니다!

### (3) 글자 요리하기 (String Methods)
문자열 내용물 뒤에 점(`.`)을 찍으면 문자를 예쁘게 다듬어주는 여러 가지 **도구(메서드)**를 불러올 수 있습니다.
- `.upper()` / `.lower()`: 글자를 모두 대문자(ABC)/소문자(abc)로 바꿉니다.
- `.replace("A", "B")`: "A"라는 글자를 찾아서 모두 "B"로 바꿉니다. (예: `text.replace("e", "a")` -> `"Hallo"`)
- `.split()`: 주로 큰따옴표 안의 문장을 공백이나 특정 기준을 통해 쪼개줍니다. 나누어진 단어들은 리스트라는 곳에 쏙 들어갑니다.
- `len(text)`: 텍스트의 전체 길이(글자 수)를 알려줍니다. (length의 줄임말입니다.)

### (4) 궁극의 요술지팡이, f-string (문자열 포매팅)
문장을 출력할 때, 내 변수를 넣고 싶을 때가 많죠? 그럴 땐 항상 앞쪽에 `f`를 붙이고 내용물 안에 중괄호 `{}`를 쓰세요. 그곳에 변수를 넣을 수 있습니다.
`name = "John"`
`print(f"My name is {name} and it is nice to meet you!")` -> 화면에 진짜 내 이름이 들어간 예쁜 문장이 출력됩니다!
