# 3주차 실습 상호평가 기준표 (Peer-Eval Rubric)

동료의 실습(Lab) 과제 캡처 화면을 보고 아래 3가지 문항에 대해 빠르고 객관적으로 각각 Yes/No 판정을 내려주세요. (각 문항당 Yes인 경우 1점, 총 3점 만점)

1. **[기능 구현 기본]** Colab 환경에서 `input()` 함수를 사용하여 2가지 이상의 값을 오류 메시지(SyntaxError 등) 없이 정상적으로 입력받아 잘 출력하였는가? (Yes 1 / No 0)
2. **[문자열 포매팅 활용]** 텍스트를 결합할 때 단순 덧셈(+) 기호나 쉼표(,)만 쓴 것이 아니라, 이번 주차의 핵심 인 `f-string` 기법 (문자열 앞 `f` 추가 및 내부 변수 `{}`) 표기법을 사용하여 최종 문장 출력을 완성하였는가? (Yes 1 / No 0)
3. **[문자열 조작 시도]** 과제 진행 과정에서 텍스트의 인덱싱(순서 끄집어내기), 슬라이싱(자르기), 또는 파이썬의 문자열 메서드(`.upper()`, `.replace()` 등) 중 하나라도 창의적으로 도입하여 코드 조작을 수행하였는가? (Yes 1 / No 0)

*(강사 특별 가이드라인: 본 평가는 학생 논리의 흐름을 확인하는 작업입니다. 코드 내부의 영어/한국어 사용은 감점 대상이 되지 않습니다. 오직 3가지 핵심 기능의 달성 여부만 확인해 주세요. 특히 2번 항목에서 f-string 대신 과거 포매팅을 쓴 경우는 No로 판별하여 서로 피드백할 수 있게 유도하세요.)*
