## [S1] 변수와 네이밍 규칙

### Slide 3: 변수란 무엇인가?
- **개념**: 데이터를 담아두는 '투명 상자'
- **할당(=)**: 오른쪽의 값을 왼쪽 상자에 집어넣는 동작
  - 예시: `age = 20`, `name = "John"`
- **재할당**: 언제든 상자 안의 내용물을 바꿀 수 있음

### Slide 4: 복합 대입 연산자 & 네이밍 규칙
- **복합 대입 연산자**: 코드를 줄여주는 마법
  - `score += 10` (기존 점수에 10 더하기)
- **파이썬의 이름표 달기 (Snake Case)**:
  - 영문자 시작, 띄어쓰기 대신 언더스코어(`_`) 사용
  - 뱀이 기어가는 모양: `student_name`
  - *(참고)* 한글 변수명도 사용할 수는 있으나 완전 자유롭지는 않음

---

## [S2] 문자열 심화 및 조작

### Slide 5: 문자열 인덱싱 (위치 찾기)
- **핵심 규칙**: 파이썬은 무조건 **0부터** 셉니다!
- `text = "Hello"`
  - `text[0]` -> 'H'
  - `text[1]` -> 'e'
- **마이너스 인덱스**: 뒤에서부터 셀 때는 -1부터 시작
  - `text[-1]` -> 'o' (맨 뒷글자)

### Slide 6: 문자열 슬라이싱 (범위 자르기)
- **문법**: `[시작번호 : 끝나는번호직전 : 보폭]`
  - 예시: `text[0:4]` -> 0번부터 3번까지만 자름
- **보폭(Step)**: 건너뛰기 
  - `text[0:5:2]` -> 0번, 2번, 4번 추출 (2칸씩)

### Slide 7: 문자열 조작 메서드 (마법 도구 모음)
- 문자열 뒤에 점(`.`)을 찍어 사용:
  - `.upper()` / `.lower()`: 대/소문자 변환
  - `.replace("A", "B")`: "A"를 "B"로 교체
  - `.strip()`: 양쪽 끝의 지저분한 공백 제거
  - `.split()`: 공백 기준으로 텍스트 형태소 분리
  - `len(text)`: 전체 문자 개수 확인

## [S3] f-string 활용 및 최종 실습

### Slide 8: f-string 도입 배경과 사용법
- 덧셈(+)이나 쉼표(,) 포매팅의 복잡함과 한계점
- **f-string 마법**: 문자열 시작점 앞 `f`, 변수는 내부 `{}`에 삽입
  - 예시: `f"{name} 님의 나이는 {age}세 입니다."`
- 영문/한국어 무관 직관적으로 이해하기 쉬운 문장 작성법

### Slide 9: 라이브 코딩 및 과제 안내
- **실습 안내**:
  - 사용자에게 나이와 이름을 `input()`으로 받아 f-string으로 명함 출력하기
  - 에러 메시지는 '조언'임을 항상 명심할 것!
