## 자율 실습 및 과제 안내 (강의 외 별도 진행)

> **안내:** 이 단계의 내용은 75분 순수 강의 시간에 포함되지 않으며, 학생 개별 자율 학습이나 별도 채널을 통한 과제로 전환됩니다.

### 1. 코딩 실습 가이드 (Hands-on)
1. **Google Colab 접속 및 활성화**
   - 학생들은 각자 브라우저에서 Colab에 접속하여 빈 셀에 `print("안녕하세요!")` 등을 실행해봅니다.
2. **GitHub 계정 생성 및 세팅**
   - 앞으로의 포트폴리오를 관리할 GitHub 계정과 과제 레포지토리를 만들고 강사를 초대합니다.
3. **AI 파트너십 구축**
   - Gemini나 ChatGPT에 가입하여 AI 코딩 파트너와 첫 인사를 나눕니다.

### 2. 마이크로 과제 및 AI 비평 (AI Review)
- 학생 스스로 AI에게 던진 질문과 AI 답변을 리뷰하며 "AI의 설명이 논리적인가? 틀릴 수도 있지 않은가?"를 토론 게시판 등을 통해 공유합니다.
- **Micro-Assignment**: 강의에서 안내된 비즈니스 이메일 5원칙을 지켜 자기소개 이메일을 제출하고, GitHub 기초 세팅 URL을 과제란에 업로드합니다.

## 강사 유의 사항

- 이 수업은 '코딩 지식 전달'보다 **수업 방식 설명**이 더 중요한 주차다. Python 문법보다 Flipped Learning과 AI 파트너 개념 정착에 집중한다.
- GitHub 계정 생성 중 인증 메일 발송이 오래 걸릴 수 있다. 안 되면 일단 넘어가고 다음 주까지 완료하도록 안내한다.
- "코딩 못 하면 이 수업 어렵나요?" 질문이 나올 수 있다. "이 수업은 코딩을 외우는 수업이 아닙니다. AI가 코드를 짜주고, 여러분은 그게 맞는지 판단하는 능력을 키웁니다"라고 답한다.
- 비즈니스 이메일 과제는 단순한 첫 과제가 아니라 **한 학기 내내 쓸 소통 방식의 기준**이다. 제출 기한과 형식을 명확히 안내한다.
