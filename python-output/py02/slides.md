## [Slide 3: Section 1 - print() 기본 사용법]
- **메인 카피**: `print()` 함수
- **텍스트**:
  - 컴퓨터에게 내리는 첫 번째 명령
  - "괄호 안에 적은 내용을 화면에 보여줘!"

  ```python
  print("안녕하세요, 파이썬!")

  print("나이:", 20)
  ```
- **비주얼**: `print`라는 단어가 스포트라이트를 받는 애니메이션

## [Slide 4: 콤마(,) vs 더하기(+) — 여러 값 출력]
- **메인 카피**: 같아 보이지만 다르다
- **코드 비교 박스**:
  - `print("이름:", "정원혁")` → **이름: 정원혁** (콤마: 공백 자동 삽입)
  - `print("이름:" + "정원혁")` → **이름:정원혁** (+: 공백 없이 이어붙이기)
- **텍스트 강조**: 콤마는 각 값 사이에 공백을 넣어준다. `+`는 문자열끼리만 이어붙인다.
- **비주얼**: 두 출력 결과를 나란히 비교하는 표

---

## [Slide 5: end= 와 sep= 옵션]
- **메인 카피**: print()를 더 자유롭게
- **코드 블록**:
  ```python
  print("A", "B", "C", sep="-")      # A-B-C
  print("로딩 중", end="...")         # 로딩 중...
  print("완료")                       # (같은 줄에 이어서) 로딩 중...완료
  ```
- **텍스트**:
  - `sep=` : 값 사이에 끼워 넣을 구분 문자 지정 (기본값: 공백)
  - `end=` : 출력 마지막에 붙일 문자 지정 (기본값: 줄바꿈 `\n`)
- **비주얼**: sep/end 작동 원리를 보여주는 도식

## [Slide 6: 주석(#) — 코드 속 메모]
- **메인 카피**: 컴퓨터가 무시하는 줄
- **코드 블록**:
  ```python
  # 이 줄은 실행되지 않습니다
  print("Hello")  # 출력 명령입니다
  ```
- **텍스트**:
  - `#` 이후의 모든 내용은 파이썬이 읽지 않는다
  - 코드에 설명을 달거나, 실행하지 않을 코드를 임시로 끄는 용도
- **비주얼**: `#` 기호 뒤가 회색 처리된 코드 에디터 화면

## [Slide 7: Section 2 - 4가지 기본 타입]
- **메인 카피**: 데이터에도 종류가 있다
- **표**:

  | 타입 | 이름 | 예시 |
  |:-----|:-----|:-----|
  | `str` | 문자열 | `"안녕"`, `'hello'` |
  | `int` | 정수 | `10`, `-3`, `0` |
  | `float` | 실수(소수) | `3.14`, `-0.5` |
  | `bool` | 참/거짓 | `True`, `False` |

- **비주얼**: 4가지 타입을 색깔별로 구분한 카드 레이아웃

## [Slide 8: type() 함수 — 타입 확인하기]
- **메인 카피**: 이 값은 무슨 타입일까?
- **코드 블록**:
  ```python
  print(type("안녕"))    # <class 'str'>
  print(type(10))        # <class 'int'>
  print(type(3.14))      # <class 'float'>
  print(type(True))      # <class 'bool'>
  ```
- **텍스트**: 모르겠으면 `type()`에게 물어보면 된다
- **비주얼**: type() 결과가 말풍선으로 나타나는 일러스트

## [Slide 9: 산술 연산자 7종]
- **메인 카피**: 파이썬의 계산기
- **표**:

  | 연산자 | 의미 | 예시 | 결과 |
  |:------:|:-----|:-----|:----:|
  | `+` | 덧셈 | `10 + 3` | `13` |
  | `-` | 뺄셈 | `10 - 3` | `7` |
  | `*` | 곱셈 | `10 * 3` | `30` |
  | `/` | 나눗셈 (실수) | `10 / 3` | `3.333...` |
  | `//` | 정수 나눗셈 | `10 // 3` | `3` |
  | `%` | 나머지 | `10 % 3` | `1` |
  | `**` | 거듭제곱 | `2 ** 10` | `1024` |

- **비주얼**: 계산기 이미지에 각 연산자 강조 표시

---

## [Slide 10: 데이터 연산의 차이]
- **메인 카피**: 타입에 따라 결과가 다르다
- **코드 비교 박스**:
  - `print(10 + 20)` → **30** (산술 덧셈)
  - `print("10" + "20")` → **1020** (문자열 이어붙이기)
  - `print("파이썬" * 3)` → **파이썬파이썬파이썬** (반복 복사)

## [Slide 11: Section 3 - input() 함수]
- **메인 카피**: 사용자의 말을 듣는 법
- **코드 블록**:
  ```python
  name = input("이름을 입력하세요: ")
  print("안녕하세요,", name)
  ```
- **텍스트**:
  - `input()`은 실행 중 멈추고 키보드 입력을 기다린다
  - 입력받은 값은 항상 **문자열(`str`) 타입**으로 돌아온다
- **비주얼**: 터미널에서 커서가 깜빡이며 입력 대기하는 화면

---

## [Slide 12: 타입 변환 함수]
- **메인 카피**: 타입을 바꾸는 마법
- **코드 블록**:
  ```python
  age_str = input("나이를 입력하세요: ")  # "20" (str)
  age_int = int(age_str)                  # 20  (int)
  print(age_int + 1)                      # 21
  ```
- **표**:

  | 함수 | 변환 방향 | 예시 |
  |:-----|:----------|:-----|
  | `int()` | str/float → 정수 | `int("20")` → `20` |
  | `float()` | str/int → 실수 | `float("3.14")` → `3.14` |
  | `str()` | int/float → 문자열 | `str(100)` → `"100"` |

- **비주얼**: 변환 화살표가 그려진 타입 변환 흐름도

## [Slide 15: SyntaxError vs TypeError]
- **메인 카피**: 에러도 종류가 있다
- **표**:

  | 에러 종류 | 원인 | 예시 |
  |:----------|:-----|:-----|
  | `SyntaxError` | 문법 자체가 틀림 (따옴표 미닫힘, 괄호 누락 등) | `print("안녕)` |
  | `TypeError` | 타입이 맞지 않는 연산 (str + int 등) | `print("나이: " + 20)` |

- **텍스트**: 에러 메시지를 읽으면 어디가 문제인지 알 수 있다 → AI에게 복사해서 물어보기
- **비주얼**: 두 에러 메시지를 나란히 보여주는 Colab 화면 캡처
