### print() 기본 사용

파이썬을 배우기 시작하면 가장 먼저 만나는 함수가 바로 `print()`입니다. 지난주에도 잠깐 써봤죠? 오늘은 이 `print()` 하나를 완전히 뜯어보겠습니다. 생각보다 훨씬 다양한 기능이 있거든요.

우선 가장 기본적인 형태입니다. Colab 코드 셀에 이렇게 입력해 보세요.

```python
print("안녕하세요, 파이썬!")
```

Shift+Enter를 누르면 셀 아래에 `안녕하세요, 파이썬!` 이라는 텍스트가 출력됩니다. 간단하죠?

`print`는 영어로 '인쇄하다', '출력하다'는 뜻입니다. 파이썬에서 `print()` 함수는 괄호 안에 넣어준 것을 화면에 출력해주는 역할을 합니다. 가장 기본적인 출력 도구이고, 여러분이 이번 학기 동안 코드를 짤 때마다 수백 번은 사용하게 될 함수입니다.

그런데 왜 따옴표가 필요할까요? `print(안녕하세요)` 라고 하면 안 되나요? 해봅시다.

```python
print(안녕하세요)
```

실행하면 에러가 납니다. 왜냐하면 파이썬은 따옴표가 없는 `안녕하세요`를 글자 그 자체가 아니라, 어딘가에서 정의된 변수 이름이라고 생각하거든요. `안녕하세요`라는 변수가 정의된 적이 없으니 에러가 나는 겁니다. 따옴표를 쓰면 파이썬이 "아, 이건 그냥 문자 데이터구나"라고 인식합니다. 이 개념은 S2에서 더 자세히 다룹니다.

숫자는 따옴표 없이 바로 출력할 수 있습니다.

```python
print(42)
print(3.14)
```

숫자는 파이썬이 데이터 자체로 바로 인식하기 때문에 따옴표가 필요하지 않습니다.

### 여러 값 동시 출력 — 콤마(,)의 힘

이제 여러 값을 한 번에 출력하는 방법을 배워봅시다. `print()` 괄호 안에 쉼표(`,`)로 값들을 나열하면 됩니다.

```python
print("나는", "파이썬을", "배우고 있다")
```

출력 결과: `나는 파이썬을 배우고 있다`

각 단어 사이에 자동으로 공백이 하나씩 들어간 것을 확인하셨나요? 파이썬에서 `print()`에 콤마로 값을 나열하면 기본적으로 각 값 사이에 공백을 한 칸 넣어서 출력해줍니다.

그렇다면 숫자와 문자를 섞어서 출력하고 싶을 때는 어떻게 할까요?

```python
나이 = 20
print("나의 나이는", 나이, "살입니다.")
```

이렇게 하면 `나의 나이는 20 살입니다.` 라고 출력됩니다. 콤마를 쓰면 문자와 숫자를 자유롭게 섞어서 출력할 수 있습니다.

---

### 콤마(,) vs 더하기(+): 핵심 차이

자, 이제 아주 중요한 부분입니다. 여러 값을 이어붙일 때 콤마(`,`) 대신 더하기(`+`)를 쓰면 어떻게 될까요? 직접 비교해봅시다.

```python
print("안녕" + "하세요")
```

출력 결과: `안녕하세요`

더하기를 쓰면 두 문자열이 **공백 없이** 바짝 붙어서 출력됩니다. 반면 콤마를 쓰면 자동으로 공백이 들어가죠. 이게 첫 번째 차이입니다.

더 중요한 차이가 있습니다. 더하기는 **문자열끼리만** 연결이 가능합니다. 문자열과 숫자를 더하기로 연결하려고 하면 에러가 납니다.

```python
나이 = 20
print("나의 나이는 " + 나이 + "살입니다.")
```

이렇게 하면 `TypeError: can only concatenate str (not "int") to str` 라는 에러가 납니다. `나이`가 숫자(int)인데 문자열(str)에 더하기(`+`)로 붙이려 했기 때문입니다.

반면 콤마를 사용하면 이런 에러가 발생하지 않습니다.

```python
나이 = 20
print("나의 나이는", 나이, "살입니다.")
```

이건 완벽하게 동작합니다. 콤마는 값들을 차례로 출력하는 방식이라, 숫자든 문자든 상관없이 처리가 가능하거든요.

**정리:**
- **콤마(`,`)**: 값 사이에 공백 자동 삽입 / 타입이 달라도 OK
- **더하기(`+`)**: 공백 없이 바짝 붙임 / 문자열끼리만 사용 가능

초보자는 처음엔 콤마를 주로 쓰는 것이 훨씬 안전합니다. 더하기로 인한 TypeError가 초보 때 정말 자주 나오거든요.

---

### end= 옵션: 줄바꿈을 바꿔보자

이제 `print()` 함수의 숨겨진 기능을 배워볼 차례입니다. 먼저 `end=` 옵션입니다.

기본적으로 `print()` 함수는 출력한 다음 줄바꿈을 자동으로 넣어줍니다. 그래서 두 줄의 `print()`를 쓰면 결과도 두 줄로 나오죠.

```python
print("첫 번째 줄")
print("두 번째 줄")
```

출력:
```
첫 번째 줄
두 번째 줄
```

`end=` 옵션을 사용하면 이 줄바꿈 대신 다른 문자를 넣을 수 있습니다. 기본값은 `end="\n"` 즉 줄바꿈 문자입니다. 이걸 바꿔볼게요.

```python
print("첫 번째 줄", end=" ")
print("두 번째 줄")
```

출력: `첫 번째 줄 두 번째 줄`

`end=" "`라고 하면 출력 끝에 줄바꿈 대신 공백 한 칸을 넣습니다. 그래서 두 `print()`의 결과가 같은 줄에 이어져 나오게 됩니다.

```python
print("로딩 중", end="...")
print("완료!")
```

출력: `로딩 중...완료!`

재밌죠? `end=`에는 어떤 문자든 넣을 수 있습니다. 빈 문자열 `end=""`를 넣으면 줄바꿈도 없고 추가 문자도 없이 바로 이어집니다.

```python
print("A", end="")
print("B", end="")
print("C")
```

출력: `ABC`

이렇게 여러 `print()` 결과를 한 줄에 이어서 출력하고 싶을 때 유용합니다.

### sep= 옵션: 구분자를 바꿔보자

`sep=`는 separator(구분자)의 줄임말입니다. 콤마로 나열한 값들 사이에 어떤 문자를 삽입할지 지정하는 옵션입니다.

앞서 콤마를 쓰면 값들 사이에 공백이 들어간다고 했죠? 그게 기본값이 `sep=" "` 즉 공백 한 칸이기 때문입니다. 이걸 원하는 문자로 바꿀 수 있어요.

```python
print("2026", "03", "07", sep="-")
```

출력: `2026-03-07`

날짜 형식처럼 구분자가 필요할 때 아주 유용합니다.

```python
print("사과", "바나나", "딸기", sep=", ")
```

출력: `사과, 바나나, 딸기`

```python
print("A", "B", "C", sep="→")
```

출력: `A→B→C`

`sep=""`로 하면 구분자 없이 값들이 바짝 붙어서 출력됩니다.

```python
print("Hello", "World", sep="")
```

출력: `HelloWorld`

`end=`와 `sep=`를 함께 사용하는 것도 물론 가능합니다.

```python
print("사과", "바나나", sep=" | ", end="!\n")
```

출력: `사과 | 바나나!`

이렇게 `sep=`와 `end=` 두 옵션을 잘 활용하면 출력 형식을 아주 다양하게 조절할 수 있습니다. 처음에 다 외울 필요는 없어요. "아, 콤마로 찍은 값들 사이에 구분자를 내가 바꿀 수 있구나" 정도만 알아두고, 필요할 때 찾아서 쓰면 됩니다.

### 주석(#): 코드 안에 메모 남기기

마지막으로 주석에 대해 알아보겠습니다. 주석은 코드 안에 적는 메모입니다. `#` 기호 뒤에 적는 모든 텍스트는 파이썬이 실행할 때 완전히 무시합니다. 파이썬 입장에서는 그 줄에 없는 것과 마찬가지입니다.

```python
# 이것은 주석입니다. 실행되지 않아요.
print("안녕하세요!")  # 이 줄은 실행됩니다
# print("이건 주석 처리된 코드라 실행 안 됩니다")
```

출력: `안녕하세요!`

주석의 용도는 크게 세 가지입니다.

**첫째, 코드 설명.** 나중에 이 코드를 다시 볼 때(혹은 다른 사람이 볼 때) 이 코드가 무슨 역할을 하는지 이해하기 쉽도록 설명을 달아놓는 겁니다.

```python
# 사용자 나이를 출력하는 코드
나이 = 20
print("나의 나이는", 나이, "살입니다.")
```

**둘째, 코드 임시 비활성화.** 특정 줄이 제대로 동작하지 않을 때, 그 줄을 삭제하지 않고 주석으로 처리해서 일단 실행에서 제외시키는 거죠. 이렇게 하면 나중에 다시 살리기도 쉽습니다.

```python
# print("이건 지금 테스트 중이라 꺼둠")
print("이건 정식 출력")
```

**셋째, 힌트 메모.** 여러분이 받게 될 실습 파일에도 주석 형태로 "여기에 코드를 작성하세요" 같은 안내가 적혀 있는 경우가 많을 거예요.

주석은 좋은 코딩 습관의 기본입니다. 코드가 아무리 짧아도, 나중에 다시 봤을 때 "내가 왜 이렇게 짰지?" 싶을 때가 반드시 옵니다. 습관적으로 주석을 달아두세요.

## Section 2: 4가지 기본 타입 + 산술 연산자 (25분)

안녕하세요, 2주차 두 번째 섹션입니다. 앞에서 `print()`로 다양한 데이터를 출력하는 방법을 배웠는데요, 이번 섹션에서는 파이썬에서 다루는 데이터의 '종류', 즉 **타입(Type)**에 대해 배웁니다.

타입이 왜 중요할까요? 우리가 일상에서 사용하는 데이터도 종류가 다 다릅니다. 이름은 문자, 나이는 숫자, 시험에 합격했는지 여부는 참/거짓. 이처럼 데이터의 종류에 따라 할 수 있는 일이 달라집니다. 파이썬도 마찬가지입니다. 데이터의 타입에 따라 어떤 연산이 가능한지, 어떻게 처리해야 하는지가 달라집니다.

파이썬의 기본 타입은 크게 네 가지입니다. `str`, `int`, `float`, `bool`. 지금부터 하나씩 살펴보겠습니다.

---

### str — 문자열

첫 번째는 **str(스트링, String)**입니다. '문자열'이라고 번역하는데, 쉽게 말하면 '글자들의 모음'입니다. 한 글자도 str이고, 문장도 str이고, 빈 문자열도 str입니다.

작은따옴표(`'`) 또는 큰따옴표(`"`) 사이에 넣은 모든 것이 문자열입니다.

```python
이름 = "정원혁"
인사말 = '안녕하세요!'
빈_문자열 = ""
```

주의할 점은, 숫자처럼 보여도 따옴표로 감싸면 문자열이 됩니다.

```python
숫자처럼_보이는_문자열 = "42"
진짜_숫자 = 42
```

`"42"`와 `42`는 완전히 다른 데이터입니다. 앞의 것은 문자열이고, 뒤의 것은 숫자입니다. 이 차이를 반드시 기억해야 합니다. 왜냐하면 `"42" + "1"`은 `"421"`이 되지만, `42 + 1`은 `43`이 되거든요.

문자열로 할 수 있는 것들이 꽤 있습니다. 더하기로 이어붙이기, 곱하기로 반복하기 등이 가능하죠.

```python
print("안녕" + " " + "세상")   # 안녕 세상
print("하하" * 3)              # 하하하하하하
print("=" * 20)               # ====================
```

마지막 예시처럼 구분선을 만들 때 문자 반복이 자주 활용됩니다.

---

### int — 정수

두 번째는 **int(인트, Integer)**입니다. '정수'라고 번역하는데, 소수점 없는 숫자입니다. ..., -2, -1, 0, 1, 2, ... 이렇게 음의 정수, 0, 양의 정수가 모두 포함됩니다.

```python
나이 = 20
온도 = -5
학생수 = 0
큰_수 = 1000000
```

파이썬에서 `int`는 아주 큰 숫자도 문제없이 처리할 수 있습니다. 다른 언어에서는 숫자 크기에 제한이 있는 경우가 많은데, 파이썬의 `int`는 이론상 메모리가 허용하는 한 얼마든지 큰 숫자를 표현할 수 있습니다.

---

### float — 부동소수점

세 번째는 **float(플로트, Floating-point number)**입니다. 소수점이 있는 숫자입니다.

```python
키 = 175.5
원주율 = 3.14159
체온 = 36.5
```

`float`는 내부적으로 '부동소수점' 방식으로 숫자를 저장합니다. 소수점의 위치가 고정되지 않고 '떠다닌다(float)'는 의미에서 붙은 이름입니다. 이 내부 원리는 지금 단계에서 깊이 알 필요는 없습니다. 소수점이 있는 숫자는 float 타입이라는 것만 기억해두세요.

중요한 특징 하나 짚겠습니다. 파이썬에서 정수끼리 나눗셈을 하면 결과가 float로 나옵니다.

```python
print(10 / 2)   # 5.0 (float!)
print(10 / 3)   # 3.3333333333333335
```

`10`과 `2`는 모두 `int`인데, 나눗셈(`/`)의 결과는 `5.0`이라는 `float`가 됩니다. 5가 아니라 5.0이라는 점 주목하세요. 이게 `int`와 `float`의 차이를 보여주는 좋은 예시입니다.

---

### bool — 불리언

네 번째는 **bool(불, Boolean)**입니다. 참(`True`) 아니면 거짓(`False`) 딱 두 가지 값만 가집니다. 불리언이라는 이름은 수학자 조지 불(George Boole)의 이름에서 따왔습니다.

```python
로그인_상태 = True
시험_합격 = False
```

`True`와 `False`는 반드시 **첫 글자만 대문자**로 써야 합니다. `true`나 `TRUE`로 쓰면 에러가 납니다.

비교 연산의 결과도 bool입니다.

```python
print(10 > 5)   # True
print(10 < 5)   # False
print(10 == 10) # True
```

불리언은 조건문에서 아주 핵심적으로 사용됩니다. "만약 로그인_상태가 True이면..." 같은 식으로요. 4주차에 조건문을 배울 때 자세히 다룰 것이니 지금은 개념만 익혀두세요.

---

### type() 함수: 타입 확인하기

자, 이제 아주 유용한 함수를 하나 소개하겠습니다. 바로 `type()` 함수입니다. 어떤 데이터의 타입이 무엇인지 확인할 때 사용합니다.

```python
print(type("안녕하세요"))   # <class 'str'>
print(type(42))             # <class 'int'>
print(type(3.14))           # <class 'float'>
print(type(True))           # <class 'bool'>
```

결과를 보시면 `<class 'str'>`, `<class 'int'>` 이런 식으로 나옵니다. `class` 키워드는 지금 단계에서는 무시하고, 따옴표 안에 있는 이름(`str`, `int`, `float`, `bool`)만 보시면 됩니다.

`type()` 함수는 코드를 짤 때 "이 데이터가 어떤 타입이지?" 하는 의문이 생길 때마다 바로 확인할 수 있는 도구입니다. 실습에서 자주 사용할 거예요.

예를 들어 이런 경우에 유용합니다.

```python
결과 = 10 / 2
print(결과)         # 5.0
print(type(결과))   # <class 'float'>
```

`10 / 2`가 `5.0`이 나온다는 걸 앞서 배웠죠? `type()`으로 확인해보면 정말로 `float`라는 걸 직접 눈으로 볼 수 있습니다.

헷갈릴 때마다 `type()`을 써서 확인하는 습관을 들이세요. 이게 초보 시절에 정말 큰 도움이 됩니다.

---

### 산술 연산자 7종

파이썬에서 숫자를 다룰 때 사용하는 산술 연산자를 배워보겠습니다. 총 7가지입니다. 이것도 실습 파일에서 직접 쳐보면서 확인해야 합니다. 지금은 눈으로 따라오세요.

```python
a = 17
b = 5
```

이 두 변수로 7가지 연산자를 모두 실습해봅시다.

**1. 더하기 (`+`)**
```python
print(a + b)   # 22
```
가장 기본. 두 숫자를 더합니다.

**2. 빼기 (`-`)**
```python
print(a - b)   # 12
```

**3. 곱하기 (`*`)**
```python
print(a * b)   # 85
```

**4. 나누기 (`/`)**
```python
print(a / b)   # 3.4
```
결과가 항상 `float`로 나온다는 것, 기억하시죠?

**5. 정수 나누기 (`//`)**
```python
print(a // b)  # 3
```
소수점을 버리고 **몫**만 남깁니다. `17 ÷ 5 = 3 나머지 2`이니까 몫은 3.

**6. 나머지 (`%`)**
```python
print(a % b)   # 2
```
나누기를 했을 때 남는 **나머지**입니다. `17 ÷ 5 = 3`이고 나머지가 `2`이니까 결과는 2.

**7. 거듭제곱 (`**`)**
```python
print(a ** b)  # 1419857
```
`17`을 `5`번 곱한 값, 즉 17⁵입니다.

---

이 7가지 중에서 `+`, `-`, `*`, `/`는 수학 시간에 이미 익숙한 것들이죠? 새로 배울 것은 `//`, `%`, `**` 세 가지입니다.

**`//`(정수 나누기)와 `%`(나머지)는 실제 프로그래밍에서 굉장히 자주 쓰입니다.** 예를 들어 어떤 숫자가 짝수인지 홀수인지 확인할 때 `%`를 씁니다. 짝수는 2로 나눴을 때 나머지가 0이고, 홀수는 나머지가 1이거든요.

```python
숫자 = 15
print(숫자 % 2)   # 1 → 홀수
```

```python
숫자 = 14
print(숫자 % 2)   # 0 → 짝수
```

`**`(거듭제곱)은 과학·수학 계산에서 유용합니다. 예를 들어 원의 넓이를 계산할 때(`r ** 2 * 3.14`), 또는 2의 n제곱을 계산할 때 쓸 수 있습니다.

```python
print(2 ** 10)   # 1024
```

---

### 연산자 우선순위

여러 연산자가 한 식에 있을 때는 수학의 사칙연산 우선순위와 동일하게 적용됩니다. 곱하기·나누기가 더하기·빼기보다 먼저 계산됩니다. 괄호를 사용하면 우선순위를 직접 지정할 수 있습니다.

```python
print(2 + 3 * 4)    # 14 (3*4=12 먼저, 그 다음 +2)
print((2 + 3) * 4)  # 20 (괄호 안 2+3=5 먼저, 그 다음 *4)
```

헷갈릴 때는 그냥 괄호를 쓰세요. 의도를 명확하게 표현할 수 있어서 코드를 읽기도 좋아집니다.

## Section 3: input() + 타입 변환 + 에러 (25분)

안녕하세요, 2주차 마지막 섹션입니다. 지금까지 우리는 `print()`로 데이터를 화면에 출력하는 것과, 다양한 타입의 데이터를 이해하는 방법을 배웠습니다. 그런데 지금까지의 코드에는 한 가지 공통점이 있습니다. 모든 데이터가 코드 안에 미리 정해져 있다는 점이죠.

예를 들어 `나이 = 20`이라고 하면, 이 코드를 실행하는 사람의 나이가 20살이든 30살이든 항상 20이 출력됩니다. 진짜 유용한 프로그램은 코드를 실행하는 사람의 입력을 받아서 그에 맞게 동작해야 합니다. 그걸 가능하게 해주는 것이 바로 `input()` 함수입니다.

### input() 함수: 사용자에게 입력 받기

`input()` 함수는 프로그램 실행 중에 잠깐 멈추고 사용자의 입력을 기다리는 함수입니다. 사용자가 키보드로 뭔가를 입력하고 Enter를 누르면, 그 입력값을 코드에서 사용할 수 있게 됩니다.

```python
이름 = input("이름을 입력하세요: ")
print("안녕하세요,", 이름, "님!")
```

이 코드를 Colab에서 실행하면 셀 아래에 입력 창이 나타납니다. "이름을 입력하세요:" 라는 안내 문구와 함께요. 여기에 본인 이름을 적고 Enter를 누르면, 그다음 줄의 `print()`가 실행되면서 "안녕하세요, [이름] 님!" 이라고 출력됩니다.

`input()` 괄호 안에 넣은 문자열은 **프롬프트(prompt)**라고 합니다. 사용자에게 무엇을 입력해야 하는지 안내해주는 역할이에요. 생략해도 되지만, 이 안내 문구가 없으면 사용자가 뭘 입력해야 할지 모르겠죠? 항상 친절한 프롬프트를 써주는 습관을 들이세요.

---

### input()의 반환값은 항상 str — 중요!

여기서 아주 중요한 사실이 있습니다. `input()` 함수는 사용자가 무엇을 입력하든 **항상 문자열(str) 타입으로 반환**합니다.

사용자가 숫자 `42`를 입력해도, 파이썬은 그것을 문자열 `"42"`로 받습니다. 진짜 숫자 `42`가 아니에요.

직접 확인해봅시다.

```python
나이 = input("나이를 입력하세요: ")
print(type(나이))   # <class 'str'>
```

사용자가 `20`을 입력해도 `type(나이)`는 `str`이 나옵니다.

이게 왜 문제가 될까요? 입력받은 나이로 계산을 하고 싶다고 해봅시다.

```python
나이 = input("나이를 입력하세요: ")
다음_나이 = 나이 + 1
print("내년에는", 다음_나이, "살입니다.")
```

이 코드를 실행하면 `TypeError: can only concatenate str (not "int") to str` 에러가 납니다. `나이`가 문자열인데 숫자 `1`을 더하려고 했기 때문입니다.

이 문제를 해결하려면 **타입 변환**이 필요합니다.

---

### 타입 변환: int(), float(), str()

파이썬에서는 타입을 변환하는 내장 함수를 제공합니다.

**`int()` — 정수로 변환**

문자열이나 float를 정수로 변환합니다.

```python
나이_문자열 = "20"
나이_정수 = int(나이_문자열)
print(나이_정수 + 1)        # 21
print(type(나이_정수))      # <class 'int'>
```

`int("20")`은 문자열 `"20"`을 정수 `20`으로 변환합니다. 이제 숫자 연산이 가능합니다.

`input()`과 함께 사용하는 전형적인 패턴은 이렇습니다.

```python
나이 = int(input("나이를 입력하세요: "))
다음_나이 = 나이 + 1
print("내년에는", 다음_나이, "살입니다.")
```

`input()`을 바로 `int()`로 감싸는 방식입니다. 이렇게 하면 사용자가 입력한 값을 받자마자 정수로 변환합니다. 이 패턴은 앞으로 정말 자주 쓰게 될 것이니 꼭 기억해두세요.

**`float()` — 부동소수점으로 변환**

소수점이 있을 수 있는 값에는 `float()`를 사용합니다.

```python
키 = float(input("키를 입력하세요 (cm): "))
print("키는", 키, "cm입니다.")
print(type(키))   # <class 'float'>
```

`int()`와 `float()`를 구분해서 쓰는 기준은 간단합니다. 소수점이 필요한 값이면 `float()`, 정수면 `int()`를 씁니다. 나이, 학년, 개수 같은 건 `int()`이고 키, 몸무게, 온도 같은 건 `float()`이 적합합니다.

**`str()` — 문자열로 변환**

숫자를 문자열로 변환할 때 씁니다.

```python
나이 = 20
문장 = "나의 나이는 " + str(나이) + "살입니다."
print(문장)   # 나의 나이는 20살입니다.
```

숫자를 더하기(`+`)로 문자열에 연결하고 싶을 때, 숫자를 `str()`로 변환하면 됩니다. 앞서 콤마로 출력하면 이런 변환이 필요 없다고 했는데, 더하기를 꼭 써야 할 상황에서는 `str()` 변환이 필요합니다.

---

### 변환이 불가능한 경우도 있다

타입 변환이 항상 성공하는 건 아닙니다. 변환이 불가능한 경우에는 에러가 납니다.

```python
int("안녕")     # ValueError! "안녕"은 숫자로 변환할 수 없음
int("3.14")    # ValueError! 소수점이 있는 문자열은 int()로 변환 안 됨
```

`"안녕"`은 숫자가 아니니까 `int()`로 변환하려고 하면 `ValueError: invalid literal for int() with base 10: '안녕'` 에러가 납니다. `"3.14"`도 소수점 때문에 `int()` 직접 변환이 안 됩니다. 그 경우엔 먼저 `float()`로 변환한 다음에 `int()`를 써야 합니다.

```python
float("3.14")        # 3.14  (OK)
int(float("3.14"))   # 3     (float 거쳐서 int)
```

이처럼 변환 함수는 **변환이 가능한 값**에만 사용할 수 있습니다.

---

### 에러는 조언이다: SyntaxError vs TypeError

이제 에러 메시지를 읽는 방법을 배워봅시다. 코딩을 하다 보면 에러는 반드시 만납니다. 절대 피할 수 없습니다. 오히려 에러를 자주 만날수록 더 빠르게 실력이 늘어납니다. 중요한 건 에러 메시지를 두려워하지 말고, 그 안에 담긴 정보를 읽는 훈련을 하는 것입니다.

에러 메시지는 크게 두 가지 정보를 담고 있습니다. **어디서 에러가 났는지**와 **어떤 종류의 에러인지**.

오늘 배울 에러 두 가지를 비교해봅시다.

**SyntaxError — 문법 에러**

`SyntaxError`는 파이썬 문법을 어겼을 때 발생합니다. 파이썬이 코드를 실행하기 전에 먼저 읽으면서 "이 코드는 문법이 잘못됐는데?" 하는 순간에 발생합니다.

```python
print("안녕하세요)
```

이 코드를 실행하면:
```
SyntaxError: unterminated string literal (detected at line 1)
```

"문자열이 끝나지 않았어요"라는 의미입니다. 따옴표를 열었는데 닫지 않아서 발생한 에러죠. 에러 메시지 옆에 `^` 표시가 문제가 된 위치를 가리켜줍니다.

`SyntaxError`의 특징은 **코드가 실행조차 되지 않는다**는 겁니다. 문법이 잘못된 코드는 파이썬이 아예 해석을 못하기 때문입니다.

**TypeError — 타입 에러**

`TypeError`는 코드 문법은 맞지만, 서로 맞지 않는 타입의 데이터를 연산하려 할 때 발생합니다. 실행 중에 발생하는 에러입니다.

```python
나이 = input("나이: ")    # 사용자가 20을 입력했지만 결과는 str
다음_나이 = 나이 + 1      # str + int → TypeError!
```

에러 메시지:
```
TypeError: can only concatenate str (not "int") to str
```

"문자열에는 문자열만 이어붙일 수 있는데, 여기서 정수(int)를 붙이려 했습니다"라는 의미입니다.

**두 에러의 핵심 차이:**

| 에러 | 발생 시점 | 원인 |
|:-----|:---------|:-----|
| `SyntaxError` | 실행 전 | 파이썬 문법을 어긴 경우 |
| `TypeError` | 실행 중 | 타입이 맞지 않는 연산을 시도한 경우 |

`SyntaxError`는 "코드 자체가 문법적으로 잘못됐어"이고, `TypeError`는 "코드 문법은 맞는데 실행하다 보니 타입이 안 맞았어"인 겁니다.

### AI와 함께 에러 해결하기

에러가 발생했을 때, 이 에러 메시지를 AI에게 가져가면 아주 빠르게 해결 방법을 찾을 수 있습니다. 방법은 간단합니다.

**에러 메시지와 코드를 함께** 보여주는 것이 포인트입니다. 코드 없이 에러 메시지만 주면 AI가 정확한 상황을 파악하기 어렵습니다.

그리고 AI의 답변을 받으면 그냥 복사해서 쓰지 마세요. AI가 왜 이런 해결책을 제안했는지, 그 논리를 이해한 다음에 직접 코드를 수정하는 훈련을 해야 합니다. 이것이 바로 이번 과제의 핵심 요구사항이기도 합니다.

### S3 정리 + 전체 2주차 마무리

이번 섹션에서는 `input()` 함수로 사용자 입력을 받는 방법, `input()`이 항상 문자열을 반환한다는 중요한 사실, 그리고 `int()`, `float()`, `str()` 로 타입을 변환하는 방법을 배웠습니다. 마지막으로 `SyntaxError`와 `TypeError`의 차이와, AI를 활용해서 에러를 해결하는 방법도 알아봤습니다.

2주차 강의 전체를 한 번 정리해봅시다.

- **S1**: `print()`로 다양한 방식으로 출력하기. 콤마와 더하기의 차이. `end=`, `sep=` 옵션. 주석(`#`)의 역할.
- **S2**: 4가지 기본 타입 `str`·`int`·`float`·`bool`. `type()` 함수로 타입 확인. 산술 연산자 7종(`+`·`-`·`*`·`/`·`//`·`%`·`**`).
- **S3**: `input()`으로 입력받기. 반환값이 항상 `str`임. `int()`, `float()`, `str()` 타입 변환. `SyntaxError` vs `TypeError`.
