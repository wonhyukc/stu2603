## 학습 목표

1. `print()`로 문자열·숫자·여러 값을 다양한 형식으로 출력한다.
2. `str`·`int`·`float`·`bool` 4가지 기본 타입과 산술 연산자 7종을 구별한다.
3. `input()`으로 사용자 입력을 받고 타입 변환(`int()`·`str()`·`float()`)을 적용한다.

## 섹션별 진행 체크리스트

### S1 — `print()` 완전 정복 (25분)

- [ ] 2주차 인사 + 1주차 복습 요약 (2분)
- [ ] `print()` 기본: 문자열 출력, 숫자 출력, 여러 값 동시 출력 설명 (5분)
- [ ] 콤마(`,`)와 `+` 연산자의 차이점 강조 (6분)
  - 콤마: 공백 자동 삽입, 서로 다른 타입 혼합 가능
  - `+`: 공백 없이 직접 이어 붙임, 동일 타입끼리만 가능
- [ ] `end=` 옵션: 기본값 `\n`을 다른 값으로 변경하는 방법 (5분)
- [ ] `sep=` 옵션: 여러 값 사이의 구분자 변경 (4분)
- [ ] 주석(`#`): 코드 설명 목적, 실행 시 무시됨 설명 (3분)

*(강사 참고: 콤마 vs + 차이는 S3 TypeError와 직결되므로 충분히 강조할 것)*

---

### S2 — 4가지 타입 + 산술 연산자 (25분)

- [ ] 데이터 타입 개념 도입: 타입이 왜 필요한가 (3분)
- [ ] `str` (문자열): 따옴표 안의 텍스트 설명 (3분)
- [ ] `int` (정수): 소수점 없는 숫자 설명 (3분)
- [ ] `float` (부동소수점): 소수점 있는 숫자 설명 (3분)
- [ ] `bool` (불린): `True` / `False` 두 값만 존재 (3분)
- [ ] `type()` 함수 라이브 시연: 각 타입을 코드로 직접 확인 (4분)
- [ ] 산술 연산자 7종 라이브 데모: `+`, `-`, `*`, `/`, `//`, `%`, `**` (6분)

*(강사 참고: `//` 몫, `%` 나머지, `**` 거듭제곱은 학생들에게 생소하므로 요리·나눠먹기 등 보편적 비유로 반드시 설명)*

---

### S3 — `input()` + 타입 변환 + 에러 (25분)

- [ ] `input()` 함수 소개: 프로그램이 사용자의 말을 듣는 방법 (4분)
- [ ] 핵심 강조: **`input()`의 반환값은 항상 `str`** (5분)
  - 숫자를 입력해도 문자열로 반환됨을 코드로 직접 시연
- [ ] `int()` 변환: 문자열 → 정수 (4분)
- [ ] `float()` 변환: 문자열 → 부동소수점 (3분)
- [ ] `str()` 변환: 숫자 → 문자열 (3분)
- [ ] `SyntaxError` 설명: 문법 오류, 코드 실행 전 발생 (3분)
- [ ] `TypeError` 설명: 타입 충돌, 실행 중 발생 + `input()` 미변환 예시 (3분)

*(강사 참고: 마지막에 "에러는 파이썬이 보내는 안내문이다"라는 메시지로 심리적 장벽 낮추기로 마무리)*
