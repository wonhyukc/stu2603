### 1-1. print() 옵션 요약

| 요소 | 설명 | 예시 | 출력 결과 |
|:-----|:-----|:-----|:---------|
| `print("값")` | 기본 출력 | `print("안녕")` | `안녕` |
| 콤마(`,`) | 공백 자동 삽입, 타입 혼합 가능 | `print("나이:", 20)` | `나이: 20` |
| `+` 연결 | 공백 없이 붙임, 문자열끼리만 | `print("안녕" + "하세요")` | `안녕하세요` |
| `end=` | 출력 끝 문자 변경 (기본: `\n`) | `print("A", end="")` | 줄바꿈 없이 `A` |
| `sep=` | 값 사이 구분자 변경 (기본: 공백) | `print("a","b", sep="-")` | `a-b` |
| `#` 주석 | 실행되지 않는 설명 메모 | `# 이 줄은 무시됨` | (출력 없음) |

### 1-2. 4가지 기본 자료형 참고표

| 타입 | 이름 | 예시 | 특징 |
|:----:|:-----|:-----|:-----|
| `str` | 문자열 | `"안녕"`, `"010-1234"` | 따옴표로 감싼 텍스트 |
| `int` | 정수 | `42`, `-5`, `0` | 소수점 없는 숫자 |
| `float` | 부동소수점 | `3.14`, `175.5` | 소수점 있는 숫자 |
| `bool` | 불린 | `True`, `False` | 참/거짓, 대문자 시작 필수 |

- `type(값)` 으로 타입을 즉시 확인할 수 있습니다.
- `"42"` (따옴표 있음) vs `42` (따옴표 없음): 완전히 다른 타입입니다.

### 1-3. 산술 연산자 7종

| 연산자 | 이름 | 예시 | 결과 | 비고 |
|:------:|:-----|:-----|:----:|:-----|
| `+` | 덧셈 | `10 + 3` | `13` | |
| `-` | 뺄셈 | `10 - 3` | `7` | |
| `*` | 곱셈 | `10 * 3` | `30` | |
| `/` | 나눗셈 | `10 / 3` | `3.333...` | 결과는 항상 `float` |
| `//` | 정수 나눗셈 (몫) | `10 // 3` | `3` | 소수점 버림 |
| `%` | 나머지 | `10 % 3` | `1` | 짝수/홀수 판별에 활용 |
| `**` | 거듭제곱 | `2 ** 3` | `8` | 2의 3제곱 = 8 |

---

### 1-4. input() + 타입 변환 요약

**핵심 규칙**: `input()` 의 반환값은 **항상 `str`** 입니다.

| 상황 | 코드 패턴 | 설명 |
|:-----|:---------|:-----|
| 숫자 계산이 필요할 때 | `int(input("입력: "))` | 정수로 변환 후 사용 |
| 소수점 계산이 필요할 때 | `float(input("입력: "))` | 부동소수점으로 변환 |
| 숫자를 문자열에 붙일 때 | `str(숫자)` | 문자열로 변환 후 `+` 사용 |

**에러 구별**

| 에러 | 발생 시점 | 주요 원인 | 해결 방법 |
|:-----|:--------:|:---------|:---------|
| `SyntaxError` | 실행 전 | 괄호 누락, 따옴표 미완성 등 | 해당 줄 문법 재확인 |
| `TypeError` | 실행 중 | `str + int` 등 타입 충돌 | 타입 변환 함수 사용 |
