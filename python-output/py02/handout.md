## 1. 개념 참고 영역

### 4가지 기본 타입

| 타입 | 이름 | 예시 | 특징 |
|:----:|:-----|:-----|:-----|
| `str` | 문자열 | `"안녕"`, `'hello'` | 따옴표로 감싼 모든 것 |
| `int` | 정수 | `42`, `-5`, `0` | 소수점 없는 숫자 |
| `float` | 부동소수점 | `3.14`, `175.5` | 소수점 있는 숫자 |
| `bool` | 불리언 | `True`, `False` | 참/거짓 두 가지만 존재 |

> `type(값)` 으로 타입을 직접 확인할 수 있습니다.
> 예: `type("안녕")` → `<class 'str'>`

---

### 산술 연산자 7종

| 연산자 | 이름 | 예시 | 결과 |
|:------:|:-----|:-----|:----:|
| `+` | 더하기 | `17 + 5` | `22` |
| `-` | 빼기 | `17 - 5` | `12` |
| `*` | 곱하기 | `17 * 5` | `85` |
| `/` | 나누기 | `17 / 5` | `3.4` (float) |
| `//` | 정수 나누기 (몫) | `17 // 5` | `3` |
| `%` | 나머지 | `17 % 5` | `2` |
| `**` | 거듭제곱 | `2 ** 10` | `1024` |

> `/` 나누기의 결과는 항상 `float`입니다.

---

### print() 옵션 요약

| 옵션 | 기본값 | 역할 | 예시 |
|:-----|:------:|:-----|:-----|
| `end=` | `"\n"` (줄바꿈) | 출력 끝에 붙는 문자 | `print("A", end="")` |
| `sep=` | `" "` (공백) | 값 사이 구분자 | `print("a","b", sep="-")` |

> `#` 기호 뒤의 텍스트는 주석으로, 파이썬 실행 시 완전히 무시됩니다.

**콤마(`,`) vs 더하기(`+`) 요약:**
- **콤마**: 값 사이 공백 자동 삽입, 타입이 달라도 OK
- **더하기**: 공백 없이 이어붙임, 문자열끼리만 가능

---

### input() + 타입 변환 요약

| 항목 | 내용 |
|:-----|:-----|
| `input("안내문구")` | 사용자 입력을 받아 반환. **반환값은 항상 `str`** |
| `int(값)` | 문자열 또는 float를 정수로 변환 |
| `float(값)` | 문자열 또는 int를 부동소수점으로 변환 |
| `str(값)` | 숫자를 문자열로 변환 |

> 자주 쓰는 패턴: `나이 = int(input("나이: "))`

**에러 타입 구분:**
- `SyntaxError` — 파이썬 문법을 어겼을 때 (실행 전 발생)
- `TypeError` — 타입이 맞지 않는 연산을 시도했을 때 (실행 중 발생)
