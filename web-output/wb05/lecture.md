## 3. 섹션별 상세 시나리오 (스크립트급 구체성)

### [S1] 25분: CSS란? 옷 입히기 비유 및 스타일시트 연결 라이브 데모
- **오프닝 및 비유 설명 (5분)**
  - **강사 액션**: 화면에 뼈대만 있는 앙상한 "HTML 집" 이미지와 예쁘게 꾸며진 "완성된 집" 이미지를 나란히 띄운다.
  - **설명 요지**: "HTML은 뼈대이고, CSS는 웹의 인테리어 및 패션입니다. 오늘 여러분은 첫 인테리어 시공을 시작합니다."
- **인라인(Inline) 및 내부(Internal) 스타일 시연 (10분)**
  - **강사 액션**: StackBlitz 환경을 열고 텍스트 태그 `<h1 style="color: red;">`를 직접 타이핑하며 실시간으로 색상이 변하는 것을 보여준다.
  - **설명 요지**: "인라인 스타일은 급할 때 쓰지만, 방이 100개면 100번을 써야 하므로 매우 비효율적입니다."
  - **강사 액션**: `<head>` 영역에 `<style>` 태그를 열어 `h1 { color: blue; }`로 일괄 변경되는 시연을 진행.
- **외부(External) 스타일시트 연결 시연 (10분)**
  - **강사 액션**: 왼쪽 파일 트리에서 `style.css` 파일을 새로 생성한다.
  - **강사 액션**: HTML 문서를 열고 `<link rel="stylesheet" href="style.css">` 태그를 타이핑하여 두 파일을 마법처럼 연결!
  - **설명 요지**: "전문가들은 무조건 옷장(CSS 파일)을 따로 두고 관리합니다."

### [S2] 25분: 선택자 적용 실전 라이브 코딩
- **선택자 3대장 위계 설명 (10분)**
  - **설명 요지**: "누구에게 옷을 입힐지 지목하는 것이 '선택자'입니다."
  - **슬라이드 활용**: 태그(반 전체 학생), 클래스(특정 동아리 학생들), 아이디(단 한 명의 학생)의 차이를 구별하는 예제 비유 제시.
- **클래스(`.`)와 아이디(`#`) 적용 데모 (15분)**
  - **강사 액션**: `index.html`에 `<p class="title">`과 `<p id="main-title">`을 작성한다.
  - **강사 액션**: `style.css`로 넘어가 `.title { font-size: 20px; }` 와 `#main-title { color: gold; }`를 차례로 부여한다.
  - **시연 포인트**: 학생들에게 우선순위가 가장 높은 `#` 아이디 선택자가 우선하여 스타일을 덮어쓰기 하는 장면을 일부러 연출하여 극적으로 보여줌.

### [S3] 25분: 색상 오류 디버깅 데모 및 상속 토론
- **오류 상황 극적 연출 (10분)**
  - **강사 액션**: 폰트 색상을 흰색(`color: white;`)으로 주고, 배경색도 흰색(`background-color: #ffffff;`)으로 설정하여 글씨가 감쪽같이 화면에서 사라진 상황을 연출!
  - **강사 액션**: 당황한 척하며 "어? 완성된 글씨가 해킹당했나요?"라고 학생들에게 질문 투척.
  - **해결 과정 시연**: 강사가 마우스로 빈 공간을 드래그(블록 지정)하여 글씨가 숨어있음을 확인시키고, "컴퓨터는 바보같이 솔직하다"는 점을 강조하여 오류 대처에 대한 심리적 부담감을 낮춤.
- **CSS 상속(Inheritance)의 마법 (15분)**
  - **강사 액션**: 부모 최상단 태그인 `<body>`에 `font-family: sans-serif;`를 주면 하위의 모든 자식 텍스트 폰트가 동시다발적으로 바뀌는 마법 시연.
  - **설명 요지**: 부모의 유전자를 물려받는 상속 개념을 설명하고, 문제가 생겼을 때 AI(Gemini)에게 "CSS 상속의 원리 때문에 폰트가 바뀌었는데 어떻게 고쳐?"라고 구체적으로 물어보는 프롬프팅 방법 안내.

### [대학 생활 4년을 책임질 단축키 5종]

| 단축키 (Windows 기준) | 기능 | 생존 의미 |
|:---|:---|:---|
| **`Ctrl + Z`** | 실행 취소 (Undo) | 방금 지워버린 글, 이동시킨 파일을 직전 상태로 완벽히 되돌립니다. 인생의 세이브 포인트입니다. |
| **`Ctrl + C`** | 복사하기 (Copy) | 선택한 텍스트/파일을 클립보드에 복사합니다. 원본은 그대로 유지됩니다. |
| **`Ctrl + V`** | 붙여넣기 (Paste) | 복사한 것을 커서 위치에 즉시 붙여넣습니다. |
| **`Ctrl + A`** | 전체 선택 (Select All) | 문서 전체 또는 폴더 내 모든 파일을 한 번에 선택합니다. |
| **`Ctrl + S`** | 저장 (Save) | 작업 중인 파일을 저장합니다. 10분에 한 번씩 습관적으로 누르세요. |

Mac 사용자는 `Ctrl` 대신 `Command(Cmd)` 키를 사용합니다.

| 단계 | 텍스트 복사 가능할 때 | 텍스트 복사 불가능할 때 |
|:---|:---|:---|
| 1단계 | 에러창 텍스트를 마우스 드래그 후 `Ctrl+C` | 구글 렌즈 앱 켜서 모니터 화면 비추기 |
| 2단계 | [translate.google.com](https://translate.google.com/)에 `Ctrl+V` 붙여넣기 | 렌즈 앱에서 '번역' 모드 선택 |
| 3단계 | 번역 결과 + "해결법" 키워드로 구글 검색 | 번역된 내용 + "해결법" 키워드로 구글 검색 |
