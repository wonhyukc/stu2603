# 5주차: 박스 모델 (Box Model)

## 1. 수업 개요

* **주제:** CSS 박스 모델 및 요소 정렬
* **강사:** Wonhyuk William Chung
* **이메일:** wonhyukc@stu.ac.kr
* **대상:** E트랙 (웹프로그래밍) 1학년 유학생
* **학습 목표:**
  1. CSS 박스 모델(margin, border, padding) 구조를 파악한다.
  2. [Build-up] 포트폴리오 명함 요소들의 간격을 조절하여 정돈한다.
  3. 겹친 요소의 마진 상쇄 현상을 이해하고 올바르게 배치한다.

## 2. 강의 진행 순서 (75분)

모든 영상은 100% 온라인 비대면(화상) 전제로 진행됩니다.

* **[S1] 박스 모델 해부학 (25분)**
  - 유학생 맞춤형 안부 인사 (날씨 등 가벼운 주제)
  - 액자와 사진 비유를 통한 CSS 박스 모델 설명
  - 요소의 경계인 `border`, 내부 여백인 `padding`, 외부 여백인 `margin` 시연
* **[S2] 요소 정렬 라이브 데모 (25분)**
  - 패딩 10px과 마진 20px의 시각적 차이 비교
  - 브라우저 개발자 도구를 활용한 박스 크기 확인법 시연
* **[S3] 디버깅 및 마진 상쇄 현상 (25분)**
  - 버튼 두 개가 찰싹 붙어있는 버그 시연
  - 마진 부여 및 마진 상쇄(Margin Collapsing) 현상 해결 라이브 코딩

## 3. 실습 및 과제 체크리스트

* **실습 (StackBlitz 환경):** 강사의 시연을 바탕으로 자신의 명함 요소 간격을 조절. (eCampus 상호평가 제출)
* **이메일 과제 (0.5 과제):** AI에게 "margin과 padding의 차이를 10살 아이에게 설명해줘"라고 질문 후 비평 작성. (제목 형식: `과제0.5 학번`)
