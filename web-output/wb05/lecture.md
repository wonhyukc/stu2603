# 5주차 강사용 교안: CSS 기초 및 생존 기본기

| 항목 | 내용 |
|:-----|:-----|
| 주차 | 5주차 |
| 주제 | CSS 기초 및 생존 기본기 훈련 |
| 실습 환경 | **StackBlitz** |
| 사전 영상 | 제공됨 (Flipped Learning 대상) |
| 총 시간 | **75분 (25분 × 3섹션)** |

## 1. 개요 및 학습 목표
1. CSS 선택자와 속성을 적용하여 스타일을 입힌다.
2. [Build-up] 명함에 나만의 색상과 폰트 입힌다.
3. 강제 상속 및 스타일 충돌 오류를 수정한다.

*(특강 포함: "컴맹 탈출: 스스로 해결하는 기계와의 소통법")*

## 2. 강의 세션 구성 (총 75분: 25분 × 3섹션)

### [S1] CSS란? 옷 입히기 비유 / 인라인, 내부, 외부 스타일시트 차이점 라이브 데모 (25분)
- CSS(Cascading Style Sheets)란 무엇인가: 웹의 '옷 입히기'
- CSS 적용 방법 비교 (라이브 데모)
  - 인라인 (Inline): `<h1 style="...">` 직접 작성
  - 내부 요소 (Internal): `<head>` 내 `<style>` 태그 구성
  - 외부 파일 연결 (External): 별도의 `.css` 파일 구성

### [S2] 기본 스타일링 시연(컬러, 백그라운드, 폰트사이즈) / 선택자(클래스 vs 아이디) 적용 라이브 코딩 (25분)
- 폰트 사이즈 조정, 디자인 적용
- 스타일링 세부 요소 시연: 색상(`color`), 배경(`background-color`), 폰트 픽셀 조정.
- 선택자의 구별과 적용
  - 전체 범용 속성 조절
  - 단일 아이디(`#`) 선택자 데모
  - 그룹 클래스(`.`) 선택자 매핑 데모

### [S3] 글씨가 안 보이는 색상 오류(배경색=글자색) 디버깅 데모 / 상속의 개념을 AI와 토론 (25분)
- 투명화 에러 해결(글씨/배경색상 동일오류 발견과 조치)
- CSS 문법에서 스타일 상속의 기본 개념을 AI 프롬프트 질의로 비유하여 해결
- **생존 기본기 훈련 특강 연계**: 브라우저 설정(Incognito 등), 파일시스템 트리 구조 파악, 강제 팝업 대처법

## [별첨] 생존 기본기 훈련
- **크롬 브라우저 설정 변경**: 크롬 언어 변경, 시크릿 모드 설정, 계정 동기화 로그인/로그아웃.
- **물리적 입력 훈련**: 외국인 유학생 대상 영타 및 한타 자판 완치, 필수 단축키 (Ctrl+C, V, Z) 반복 체득.
- **파일 트리 구조 숙지**: 화면 바탕화면/내문서에 대한 저장 원리와 개념적 확장자 이해.
- **오류 경고 마인드셋**: 에러 메시지가 떴을 때 즉시 닫지 않고 차분히 텍스트를 파인딩하여 구글링 및 AI를 통해 원인을 발견하는 '컴퓨터와의 대화'의 심리적 장애물 격파.
