# Midterm Workshop Module Questions and Rubric / 중간고사 상호평가(Workshop) 모듈용 문제 및 루브릭

We have completely reorganized the format so that instructors can copy and paste all 20 questions into the **Moodle Workshop module** at once, avoiding the hassle of registering each question individually into the question bank. 
HTML tags have been applied to prevent line breaks (Enter) from being ignored when pasting directly into the Moodle editor.
강사님께서 20개의 문항을 일일이 문제은행에 등록하는 번거로움을 피하고, **무들 상호평가(Workshop) 모듈**에서 한 번에 복사/붙여넣어 운영하실 수 있도록 형태를 완전히 개편했습니다. 
무들의 편집기(에디터)에 바로 붙여넣었을 때 **줄바꿈(엔터)이 무시되는 현상을 방지하기 위해 HTML 태그를 적용**했습니다. 

If you click the **`</>` (HTML Source Edit)** button in the editor and copy/paste the code below, the formatting will be perfectly maintained.
편집기의 **`</>` (HTML 소스 편집)** 버튼을 누르고 아래 코드를 복사해서 붙여넣으시면 서식이 완벽하게 유지됩니다.

## 1. Instructions for submission / 제출 지시문
> **How to use / 사용 방법**: In the Workshop settings, go to 'Submission settings' -> 'Instructions for submission' editor. Open the `</>` (HTML Source Edit) window and paste the code below exactly as it is. / 상호평가 설정의 '제출 설정' -> '제출 지시문' 에디터에서 `</>` (HTML 소스 편집) 창을 열고 아래 코드를 그대로 붙여넣으세요.

```html
<h3>[Web Programming Midterm Time-Attack Mission / 웹프로그래밍 중간고사 타임어택 미션]</h3>
<p>This quiz is a time-attack mission where the use of AI (ChatGPT, Gemini, etc.) and internet search is fully allowed. Please read the following 20 questions in order and submit all your answers within the time limit. (In case of a tie, a higher score will be given to the earlier submission.)<br>
본 퀴즈는 AI(ChatGPT, Gemini 등) 및 인터넷 검색을 전면 허용하는 타임어택 퀴즈입니다. 아래 20개의 문항을 순서대로 읽고, 제한 시간 내에 모든 정답을 작성하여 제출해 주시기 바랍니다. (동점일 경우 제출 시각이 빠른 순으로 높은 점수가 부여됩니다.)</p>

<h4>[Multiple Choice / 객관식 - 10 Questions/문항]</h4>
<p><b>📌 Q1. Which attribute is used to specify alternate text to be displayed when an image fails to load or breaks when using the &lt;img&gt; tag? / &lt;img&gt; 태그를 사용할 때, 이미지가 깨지거나 로드되지 않을 경우 화면에 대신 표시될 텍스트를 지정하는 속성은 무엇인가?</b><br>
① src<br>
② href<br>
③ alt<br>
④ title</p>

<p><b>📌 Q2. Which of the following is NOT one of the '4 elements of an engineer's questioning method' used when asking questions to AI or senior developers? / AI나 선배 개발자에게 질문할 때 사용하는 '엔지니어식 질문법 4요소'에 해당하지 않는 것은?</b><br>
① Current phenomenon / 현재 발생하는 현상<br>
② Attempts made to solve the problem / 문제를 해결하기 위해 시도한 내용<br>
③ Your subjective feelings and emotions / 본인의 주관적인 감정과 느낌<br>

<p><b>📌 Q3. Inside an HTML &lt;form&gt;, what is the correct type for a button used to transmit user-input data to the server? / HTML &lt;form&gt; 내부에서, 사용자가 입력한 데이터를 서버로 전송하기 위해 사용하는 버튼의 올바른 타입(type)은 무엇인가?</b><br>
① type="button"<br>
② type="submit"<br>
③ type="reset"<br>
④ type="send"</p>

<p><b>📌 Q4. What is the correct CSS selector to select an HTML element with the id attribute value of "header"? / CSS에서 HTML 요소의 id 속성값이 "header"인 요소를 선택하기 위한 올바른 선택자는 무엇인가?</b><br>
① .header<br>
② #header<br>
③ *header<br>
④ @header</p>

<p><b>📌 Q5. In the CSS Box Model, which property refers to the inner space between the element's Content and Border? / CSS 박스 모델(Box Model)에서 요소의 콘텐츠(Content)와 테두리(Border) 사이의 안쪽 여백을 의미하는 속성은?</b><br>
① margin<br>
② padding<br>
③ border-spacing<br>
④ outline</p>

<p><b>📌 Q6. When using Flexbox layout, which property is used to align child elements to the center along the main axis (typically horizontal)? / Flexbox 레이아웃을 사용할 때, 자식 요소들을 메인 축(주축, 기본적으로 가로)을 기준으로 가운데 정렬하기 위해 사용하는 속성은?</b><br>
① align-items: center;<br>
② justify-content: center;<br>
③ text-align: center;<br>
④ vertical-align: middle;</p>

<p><b>📌 Q7. Which of the following tags is used to represent the largest and most important heading in an HTML document? / 다음 중 HTML 문서에서 가장 크고 중요한 제목을 나타낼 때 사용하는 태그는?</b><br>

<p><b>📌 Q8. Which of the following is NOT a correct shortcut to open the web browser's 'Developer Tools'? / 웹 브라우저의 '개발자 도구(Developer Tools)'를 열기 위한 단축키로 올바르지 않은 것은?</b><br>
① F12<br>
② Ctrl + Shift + I (Windows) / Cmd + Option + I (Mac)<br>
③ Ctrl + C (Windows) / Cmd + C (Mac)<br>
④ Right-click on the page and select 'Inspect' / 페이지 우클릭 후 '검사(Inspect)' 클릭</p>

<p><b>📌 Q9. What is the most appropriate reason why the buttons in the code below stack vertically instead of aligning horizontally side-by-side? / 아래 코드에서 버튼들이 나란히 가로로 배치되지 않고 세로로 쌓이는 원인으로 가장 적절한 것은?</b><br>

① Because flex-direction: column is the default. / flex-direction: column이 기본값이기 때문이다.<br>
② Because the parent element's display is not flex. / 부모 요소의 display가 flex가 아니기 때문이다.<br>
③ Because the margin: 0 property is missing. / margin: 0 속성이 빠졌기 때문이다.<br>
④ Because buttons originally cannot be aligned horizontally. / 버튼은 원래 가로로 배치할 수 없기 때문이다.</p>

<p><b>📌 Q10. What is the most correct meaning of the following code written in a CSS file? / CSS 파일에 작성된 다음 코드의 의미로 가장 올바른 것은?</b><br>

③ Changes the text color of all &lt;p&gt; tag elements in the document to blue. / 문서 내의 모든 &lt;p&gt; 태그 요소의 글씨 색상을 파란색으로 변경한다.<br>
④ Changes the text color of only the first &lt;p&gt; tag to blue. / 첫 번째 &lt;p&gt; 태그의 글씨 색상만 파란색으로 변경한다.</p>
```

## 💡 Precautions for Copy/Paste / 복사/붙여넣기 시 주의사항
1. Click the **`</>` shaped icon (HTML source editor)** on the first line of the Moodle (eCampus) editor menu. / 무들(eCampus) 편집기 메뉴에서 첫 번째 줄에 있는 **`</>` 모양 아이콘(HTML 소스 편집기)**을 클릭합니다.
2. When a black or separate source code window opens, copy and paste the provided code from `<h3>...` to `</p>` (or `</table>`). / 검은색 혹은 별도의 소스코드 창이 열리면, 위 제공된 `<h3>...` 부터 `</p>` (혹은 `</table>`) 까지의 코드를 복사해서 붙여넣습니다.
3. Close the source edit window to see that the line breaks, bold text, and table formatting have been perfectly applied. / 소스 편집창을 닫으면 줄바꿈과 볼드체, 테이블 서식이 완벽하게 적용된 것을 확인할 수 있습니다.
4. Now students can see all 20 questions at a glance and submit them within 1 Workshop module! / 이제 학생들은 1개의 상호평가 모듈 안에서 20개의 문제를 한눈에 보고 제출할 수 있습니다!
