# Peer Evaluation Rubric (11점 만점 / 11 Points Total)

| Question Number | Evaluation Item (Single Condition) | Score |
|---|---|---|
| Q1 | Was the assignment submitted before the deadline? (Yes 5 / No 0) | 5 |
| Q2 | Does the **navigation menu** use `display: flex;` for its layout? (Yes 1 / No 0) | 1 |
| Q3 | Are the navigation menu items spaced out using a Flexbox alignment property (e.g., `justify-content`)? (Yes 1 / No 0) | 1 |
| Q4 | Does the **main content area** (containing the cards) use Flexbox to arrange items side-by-side? (Yes 1 / No 0) | 1 |
| Q5 | Is the `gap` property used to manage spacing between the flex items? (Yes 1 / No 0) | 1 |
| Q6 | Is the previous fixed-width code (e.g., `width: 1000px`) for the profile image **commented out** instead of deleted? (Yes 1 / No 0) | 1 |
| Q7 | Is `max-width: 100%;` correctly applied to the profile image for responsiveness? (Yes 1 / No 0) | 1 |
