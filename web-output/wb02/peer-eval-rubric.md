## 1. 채점 기준 (각 1점, 총 3점 만점) / Grading Criteria (1 point each, Total 3 points)

평가자는 배정받은 학생의 제출 내용(텍스트)을 확인하고, 아래 세 가지 필수 항목에 대해 각각 **Yes(1점) / No(0점)** 로 단순명료하게 채점합니다.
*The evaluator will review the assigned student’s submission (text) and grade the following three essential criteria clearly as either **Yes (1 point) / No (0 points)**.*

| 번호 / No. | 평가 기준 (Yes/No 단위 채점) / Evaluation Criteria (Yes/No) | 배점 / Score | 확인 포인트 / Check Point |
|:---:|:---|:---:|:---|
| **1** | **기한 내 제출 / On-time Submission**<br>과제를 기한 내에 지각하지 않고 제출하였습니까?<br>*Did they submit the assignment within the deadline?* | **5점 / 5pts** | 구글 폼, eCampus 등의 제출 기록이 마감 기한을 넘기지 않았는지 확인합니다. 지각 시 0점입니다.<br>*Check if the submission timestamp on Google Forms, eCampus, etc., is within the deadline. 0 points for late submission.* |
| **2** | **URL에 'github' 포함 / URL contains 'github'**<br>제공된 **URL에 'github'**라는 단어가 포함되어 있는가?<br>*Does the provided URL contain the word 'github'?* | **1점 / 1pt** | 학생이 제출한 링크 주소 텍스트 안에 `github.com` 도메인이 정상적으로 포함되어 있으면 1점을 부여합니다.<br>*Award 1 point if the submitted link text properly includes the `github.com` domain.* |
| **3** | **해쉬값(hash) 포함 / Hash included**<br>제출 내용에 **해쉬값(hash)**이 포함되어 있는가?<br>*Does the submission contain a hash value?* | **1점 / 1pt** | 링크 주소를 포함하거나 별도로 작성된 텍스트 안에 길고 복잡한 영문자/숫자 조합의 해시값이 명시되어 있으면 1점을 부여합니다.<br>*Award 1 point if a long, complex alphanumeric hash value is clearly specified either within the link URL or as separate text.* |

## 2. 평가자(피드백 작성자) 특별 가이드라인 / Special Guidelines for Evaluators

- **내용 평가는 제외합니다 / Do not evaluate the content quality**: 디지털 명함 코드가 HTML 구조에 맞게 복잡하게 잘 짜여졌는지, 사진이 예쁘게 들어갔는지 등 질적인 평가는 반영하지 마십시오. 오직 **위 3가지 항목(제출함, 깃허브 주소 포함됨, 해쉬값 적혀있음)**의 충족 여부만 체크합니다.<br>*Do not penalize based on how well the digital business card code is structured or how good the photo looks. Evaluate ONLY based on the fulfillment of the three criteria above (Submitted, GitHub URL included, Hash included).*
- **만점 부여 / Give full marks for meeting criteria**: 코드가 정말 짧거나 단순히 예제만 복사한 수준이라도, 위 3가지 조건만 맞으면 감점 없이 **만점(3점)**을 주어야 합니다.<br>*Even if the code is extremely short or just copied from an example, you must award full marks (3 points) without deduction as long as the 3 conditions are met.*
- **피드백 작성 / Writing feedback comments**: 피드백 코멘트를 쓴다면 "해당 조건들이 모두 잘 확인됩니다." 와 같이 객관적 사실 여부 위주로 작성해주시면 충분합니다.<br>*If you write a feedback comment, it is sufficient to state objective facts, such as "All the required conditions are well confirmed."*
