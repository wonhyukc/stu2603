# [S1] HTML이란 무엇인가?

- **H**yper**T**ext **M**arkup **L**anguage
- 웹의 가장 핵심적인 뼈대(Skeleton)
- 복잡한 논리를 짜는 프로그래밍 언어가 아님 — 태그로 표시하는 마크업 언어

---

# 인체에 비유해봅시다

- `<html>`: 인간 그 자체 — 모든 것을 감싸는 껍데기
- `<head>`: 뇌 — 메타데이터 등 보이지 않는 설정
- `<body>`: 몸통 — 모니터에서 눈으로 보는 모든 것

---

# StackBlitz 데모

- 브라우저에서 직접 빈 HTML 프로젝트를 만들어 봅시다.
- `<h1>`과 `<p>` 태그 라이브 코딩 구동 데모.

```html
<body>
  <h1>나의 프로필</h1>
  <p>안녕하세요!</p>
</body>
```

# [S2] 이미지 넣기 (`<img>` 태그)

- 가장 중요한 미디어 태그
- **예외**: 닫는 태그가 없습니다! (빈 태그)
- 이미지의 출처를 가리키는 `src` 속성 강조

```html
<img src="이미지_주소">
```

---

# 절대경로 vs. 상대경로

| 종류 | 설명 | 예시 |
|:---|:---|:---|
| **절대경로** | 인터넷 어딘가의 전체 URL — 국제 우편 주소처럼 어디서든 동작 | `https://example.com/photo.jpg` |
| **상대경로** | 현재 파일 기준의 상대 위치 — 같은 폴더에 이미지가 있을 때 사용 | `./images/photo.jpg` |

오늘은 절대경로(`https://`)를 사용합니다.

# 마무리 — 오늘의 핵심 3가지

1. **태그 문을 항상 닫아라** — `<h1>...</h1>`
2. **이미지 경로(`src`)에 오타 = 엑박** — 절대경로를 꼼꼼히 확인
3. **에러는 적이 아니라 브라우저의 힌트** — 두려워 말고 디버깅하라
