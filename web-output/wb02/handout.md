## 1. Core Concepts Reference (개념 참고 영역)

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It provides the structure of the page, much like the skeleton of the human body.
HTML(HyperText Markup Language)은 웹 페이지를 만드는 데 사용되는 표준 마크업 언어입니다. 인체의 뼈대처럼 페이지의 구조를 잡아줍니다.

### Essential HTML Structure (필수 HTML 구조)
- `<html>`: The root element that wraps everything. (모든 것을 감싸는 최상위 요소)
- `<head>`: Contains metadata, title, and invisible settings. (메타데이터, 제목 등 보이지 않는 설정 포함)
- `<body>`: Contains the visible content of your web page. (웹 페이지 화면에 보이는 실제 내용 포함)

### Commonly Used Tags (자주 쓰는 태그)
| Tag (태그) | Description (설명) | Example (예시) |
|---|---|---|
| `<h1> ~ <h6>` | Headings. `<h1>` is the largest. (제목. `<h1>`이 가장 큼) | `<h1>Hello</h1>` |
| `<p>` | A paragraph of text. (문단) | `<p>My name is ...</p>` |
| `<img>` | Embeds an image. It is an empty tag (no closing tag). (이미지 삽입. 닫는 태그 없음) | `<img src="url">` |

## 3. Micro-Assignment (과제 섹션)

**Task:** Create and modify the basic skeleton of your online digital business card using AI, and submit the results.
**과제 내용:** AI를 활용해 나만의 온라인 디지털 명함 뼈대를 만들고 수정한 결과를 제출하세요.

### Assignment Format (제출 형식)

Your submission must fit within half an A4 page and include the following three parts:
A4 반 페이지 분량 내외로 작성하며, 반드시 아래 3가지 항목을 포함해야 합니다.

1. **AI Prompt (AI 질문표)**
   - Exactly what you asked the AI.
   - 예시: "나만의 온라인 명함 뼈대 HTML 작성해줘."
2. **AI Critique (AI 답변 비평)**
   - Analyze the AI's response in one sentence. What is correct and what is annoying/wrong?
   - 1문장으로 요약 비평. "틀이 잡혀 있어서 좋았지만, 내가 원하지 않는 과도한 스타일 태그가 섞여 있어서 코드가 복잡해 보였다."
3. **Final Result (최종 결과 적용본)**
   - The final code after you deleted or modified at least one tag (e.g., removing a complicated `<h2>` or `<img>` tag and putting your own). Take a screenshot or copy the code.
   - AI가 준 코드 중 마음에 들지 않는 태그 1개 이상을 직접 삭제하거나 수정한 후 캡처한 화면.
