# wb02 과정 진행 가이드 (강사용)

본 문서는 **웹프로그래밍(E트랙) 2주차** 대면 실습 및 강의 진행을 위한 강사용 가이드입니다. **Flipped Learning** 기반이므로 학생들은 사전에 제공된 영상(`lecture-script.md` 기반)을 시청하고 참여함을 전제로 합니다.

## 1. 2주차 기본 정보

| 항목 | 내용 |
|:---|:---|
| **주차** | 2주차 |
| **주제** | HTML 기초 |
| **실습 환경** | StackBlitz |
| **사전 영상** | eCampus에 업로드된 `wb02` 강의 영상 |
| **총 시간** | 75분 (25분 × 3섹션) |

## 2. 주차 학습 목표

본 주차가 끝난 후 학생이 달성해야 할 목표입니다. (마스터 교육안과 100% 일치)

1. HTML 문서 구조와 텍스트/이미지 태그(`h1`, `p`, `img`) 파악한다.
2. [Build-up] 나만의 '명함' 페이지 뼈대 조립한다.
3. 빠진 태그를 찾아 추가하는 디버깅 경험한다.

---

## 3. 섹션별 시간 배분 및 진행 시나리오

학생 실습 시간과 강사 이론/데모 시간은 철저히 구분되며, 아래 명세된 75분은 **순수 강사의 라이브 코딩 및 데모 시간** 기준입니다.

| 섹션 시간 | 연계 산출물 | 핵심 진행 시나리오 (라이브 코딩 명세) |
|:---|:---|:---|
| **[S1] 이론**<br>(25분) | `lecture-script.md` | **HTML의 뼈대 구조 및 텍스트 태그**<br>- 브라우저에 띄울 빈 StackBlitz HTML 프로젝트 생성 시연<br>- 사람의 몸에 비유하여 `<html>`, `<head>`, `<body>` 태그 구조 설명<br>- `<h1>` (머리말)과 `<p>` (문단) 태그를 타이핑하며 텍스트 렌더링 라이브 코딩 데모 |
| **[S2] 조작**<br>(25분) | `lecture-script.md`<br>`handout.md` | **이미지 태그 및 버그 수정 데모**<br>- `<img>` 태그 작성법 및 닫는 태그가 없는 이유(빈 태그) 설명<br>- `src` 속성 소개: 절대경로와 상대경로의 차이를 이미지 링크 복사붙여넣기로 시연<br>- 의도적으로 오타가 난 이미지 경로(`src`)를 입력하여 '엑스박스' 에러를 유발하고, 속성을 수정하여 이미지를 다시 띄우는 디버깅 시연 |
| **[S3] AI리뷰**<br>(25분) | `lecture-script.md`<br>마이크로 과제 | **포트폴리오 명함 조립 및 AI 비평 훈련**<br>- [Build-up] 미리 준비한 '온라인 명함 템플릿(일부 태그가 누락된 상태)' 코드를 StackBlitz에 붙여넣음<br>- 화면이 깨지는 이유를 브라우저에서 분석하고, 누락된 `<img>` 및 `<p>` 태그를 채워 넣는 해결 과정 시연<br>- AI(Gemini/ChatGPT)에게 "나만의 온라인 명함 뼈대 HTML 작성해줘"라고 요청한 후, 마음에 안 드는 태그(예: `<h2>`)를 학생이 직접 삭제/수정하여 캡처하는 과제 수행 방식 안내 |
