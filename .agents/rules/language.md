Terminology & Tone Check

AI agents and the user must strictly adhere to the following terminology in all operations. This list supplements the rules in `1prd.md`.

| Category | Type | Unified Term (Recommended) | Expressions to Avoid (Forbidden) |
|---|---|---|---|
| System | Tool | **Skill / Workflow** | Macro, Action, Script |
| Role | Persona | **Agent / Designer / Creator** | Bot, AI, Assistant |
| Learner | Reference | **Student** | Trainee, Participant, Learner (Apply flexibly if context requires) |
| Task | Format | **Weekly Assignment** | Homework, Test |

### Tone & Manner
- All documents and conversations must be **clear and concise**, utilizing a **guiding** tone rather than an imperative one.
- Use Bullet points / Numbering for complex instructions to improve readability.
