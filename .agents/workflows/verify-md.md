---
description: 마크다운(.md) 파일의 무결성을 검사하고 잔여 플레이스홀더를 확인합니다.
---

에이전트 또는 사용자가 마크다운 파일을 수정한 후, 파일 내에 실수로 남은 강사 이름 플레이스홀더 및 주요 포맷팅 오류가 없는지 검증하는 워크플로우입니다.

1. 터미널에서 `grep` 명령어를 사용하여 최근 작업한 마크다운 파일들에 치환되지 않은 플레이스홀더가 남아있는지 확인합니다.
// turbo
2. `grep -rn "\[강사 이름\]" python-output/ web-output/ || true`
// turbo
3. `grep -rn "\[이름\]" python-output/ web-output/ || true`

4. (선택 사항) 마크다운 문법(언밸런스한 코드 블록 등)을 전체적으로 검사하고 싶다면 아래 명령어를 실행합니다.
// turbo
5. `npx -y markdownlint-cli "**/*.md" --ignore node_modules`

6. 플레이스홀더가 터미널에 검색될 경우, 해당 파일을 다시 열어 강사의 실제 이름과 이메일로 치환하는 수정 작업을 진행합니다.
