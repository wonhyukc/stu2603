# eCampus 전원 출석 자동 처리 워크플로우

eCampus 출석 관리 페이지에서 매주 수작업으로 "전원 출석"을 처리하는 작업을 브라우저 에이전트에게 위임하여 자동화합니다. 사용자가 지정한 강좌의 3개 교시(예: Python 1, 2, 3교시)에 대해 자동으로 반복 수행합니다.

## 사용 방법

아래 프롬프트를 복사하여 채팅창에 붙여넣고 `[ ]` 안의 내용을 본인의 상황에 맞게 변경하여 실행하세요.

```text
/ecampus-attendance
접속 URL: [해당 강좌의 Offline attendance 페이지 URL]
대상 교시: [1, 2, 3] (또는 웹반의 경우 5, 6, 7 / 8, 9, 10)
```

## 에이전트 동작 순서 (AI's Background Behavior)

지시를 받은 AI는 사용자의 로그인 세션을 통해 다음 작업을 순차적으로 자동 수행합니다.

1. **지정된 URL 접속**: eCampus 강좌의 'Offline attendance' (출석 관리) 페이지로 이동합니다.
2. **주차/교시 선택**: 'Week' 옵션에서 대상 교시 배열의 첫 번째 교시(예`1Class`)를 선택합니다.
3. **목록 표시 변경(Number of list)**: `All` 로 변경하여 한 페이지에 모든 학생이 표시되도록 합니다.
4. **일괄 출석 선택**: 표 상단의 라디오 버튼 중 `Attendance(출석)` 를 선택합니다.
5. **일괄 적용**: `Batch processing status(일괄 처리 상태)` 버튼을 클릭하여 모든 학생의 상태를 '출석'으로 변경합니다.
6. **저장**: 페이지 최하단의 `Save(저장)` 버튼을 눌러 확정합니다.
7. **반복**: 위 과정을 대상 교시 배열의 나머지 교시들 (예: 2, 3교시)에 대해서도 동일하게 반복합니다.
8. 완료 후 결과를 사용자에게 보고합니다.
