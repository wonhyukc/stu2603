# OBS 활용 가이드

## 녹화 전 필수 체크리스트 (사고 방지용)

녹화를 마친 후 오디오가 비어 있거나 엉뚱한 화면이 녹화되는 참사를 막기 위해, **녹화 버튼을 누르기 전 반드시 아래 항목들을 확인**하세요.

- [ ] **마이크 활성화 확인**: OBS 하단의 '오디오 믹서' 패널에서 마이크(Mic/Aux) 볼륨 바가 내가 말할 때 초록색/노란색으로 움직이는지 확인합니다.
- [ ] **마이크 음소거 해제 확인**: 오디오 믹서의 스피커 아이콘이 빨간색(음소거)으로 되어 있지 않은지 확인합니다.
- [ ] **캡처할 화면/창 확인**: '소스 목록'에서 캡처하려는 디스플레이나 윈도우 창이 최상단에 있고 눈동자 아이콘이 켜져(가려지지 않음) 있는지 확인합니다.
- [ ] **미리보기 화면 확인**: OBS 중앙의 미리보기 캔버스에 내가 지금 녹화하려는 화면(브라우저, PPT, 코드 에디터 등)이 정확하게 보이고 잘리지 않았는지 최종 확인합니다.
- [ ] **테스트 녹화 (권장)**: 10초 정도 짧게 녹화한 후 영상을 재생하여 화면과 목소리가 모두 정상적으로 담겼는지 확인하는 것이 가장 안전합니다.
