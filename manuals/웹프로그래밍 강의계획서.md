# **[최종] 웹프로그래밍 교육 과정 / [Final] Web Programming Curriculum**

This curriculum is designed for 1st-year computer software students at Seoul Theological University, aiming to maximize practical skills by using generative AI as an engineering partner. All theories are pre-learned through online videos, and a **Flipped Learning** approach is adopted to focus on practice and critique in the classroom.
본 교육 과정은 서울신학대학교 컴퓨터소프트웨어학과 1학년 학생들을 대상으로 하며, 생성형 AI를 공학적 파트너로 활용하여 실무 역량을 극대화하는 것을 목표로 합니다. 모든 이론은 온라인 영상으로 사전 학습하고, 강의실에서는 실무와 비평에 집중하는 **거꾸로 수업(Flipped Learning)** 방식을 채택합니다.

## **3. 평가 가이드라인 / Evaluation Guidelines (Draft)**

The exam and evaluation methods are currently a **Draft** and will be confirmed during the semester based on student achievement. The basic policy is to fully allow the use of AI, but centrally evaluate the **'student's understanding and thinking process'**.
시험 및 평가 방식은 현재 **가안(Draft)**이며, 학기 진행 중 학생들의 성취도에 따라 확정될 예정입니다. 기본 방침은 AI 사용을 전면 허용하되 **'학생의 이해도와 사고 과정'**을 중점적으로 평가하는 것입니다.

* **Asynchronous video / 비동기 설명 영상:** During the final project, we verify understanding by having students submit a video directly explaining their code logic. / 기말 프로젝트 시 본인의 코드 로직을 직접 설명하는 영상을 제출받아 이해도를 검증합니다.
* **Workspace Analysis / 작업 이력 분석:** We check whether the AI's code was simply copied or modified while pondering line by line through IDE logs. / IDE 로그를 통해 AI의 코드를 단순히 복사했는지, 아니면 한 줄씩 고민하며 수정했는지 확인합니다.
* **Small Group Interview / 소그룹 면접:** We conduct 'pressure questions' asking what happens if specific parts of the AI-suggested code are changed through live sessions. / 실시간 세션을 통해 AI가 제안한 코드의 특정 부분을 바꾸면 어떻게 될지 묻는 '압박 질문'을 진행합니다.

**Instructor's Suggestion / 교수자 제언:** This course is designed because AI can take over simple memorization and repetitive coding tasks. Students learn basic grammar not as an 'object of memorization' but as a **'weapon to control AI'**, ultimately deepening their actual learning level.
**교수자 제언:** 본 과정은 AI가 단순 암기나 반복 코딩 업무를 분담하기 때문에 가능한 설계입니다. 학생들은 기본 문법을 '외우는 대상'이 아닌 **'AI를 통제하기 위한 무기'**로 배우게 되므로, 실제 학습 수준은 오히려 더 깊어집니다.
