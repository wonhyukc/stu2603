# [Week 1] Web Programming (E-Track) - Implementation Plan
# [1주차] 웹프로그래밍(E트랙) - 교안 작성 계획

## 1. 개요 (Overview)
본 계획은 웹프로그래밍(E트랙) 1주차 수업을 위한 교안 및 실습 자료 제작을 위한 가이드라인입니다. 1학년 국제학부생들을 대상으로 하므로, 기술적 용어보다는 직관적인 비유와 AI 활용 기초를 다지는 데 집중합니다.
This plan outlines the creation of teaching materials and lab guides for Week 1 of Web Programming (E-Track). Aimed at 1st-year International Students, it focuses on intuitive analogies rather than complex technical jargon, establishing the foundations of AI-native learning.

## 2. 학습 목표 (Learning Objectives)
1. **AI-Native 컴퓨팅 사고 이해**: 단순 암기 대신 AI를 파트너로 활용하는 수업 방식을 이해한다.
   * Understand AI-Native Computing Thinking: Grasp the class methodology centered on using AI as a partner rather than rote memorization.
2. **웹의 기본 원리 파악**: '문서'와 '브라우저'의 관계를 쉬운 비유로 이해한다.
   * Grasp Web Fundamentals: Understand the relationship between 'documents' and 'browsers' through simple analogies.
3. **개발 환경 구축**: 수업에 필요한 도구(VS Code, Python 등)를 설치하고 정상 동작을 확인한다.
   * Environment Setup: Install and verify essential tools (VS Code, Python, etc.) required for the course.
4. **엔지니어식 소통 실천**: 교수자 및 AI에게 질문하는 올바른 '엔지니어식 질문법'을 익힌다.
   * Practice Engineer-style Communication: Learn the proper 'engineer-style questioning method' for instructors and AI.
