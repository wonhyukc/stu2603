# 1주차 강의안: 웹과 입문 / Week 1 Lecture: Intro to Web

## 1. 수업 목표 / Course Objectives
* 인터넷 문서가 웹사이트로 변하는 과정을 구경한다.
* 브라우저 기반 실습 환경(StackBlitz) 사용법을 익힌다.
* eCampus 사용 및 엔지니어식 소통 방법을 실습한다.

## 2. 수업 개요 / Class Overview
1. 오리엔테이션: 대학 생활과 AI 협업의 기초 (Reduced Class Load)
2. 어떻게 웹사이트가 보일까? (쉬운 비유)
3. 첫 웹문서 렌더링 실습

## 3. 핵심 개념 / Core Concepts
* **Web Browser**: 문서를 예쁘게 그려주는 화가.
* **HTML**: 문서의 뼈대와 구조.
* **StackBlitz**: 클릭 한 번으로 나만의 웹 개발 실험실을 여는 마법의 열쇠.
