# 1주차 실습: StackBlitz 첫 경험 / Week 1 Lab: First StackBlitz Experience

## 목표 / Objective
* 브라우저 기반의 StackBlitz 환경에 접속하고 기본 구조를 살펴본다.
* 복잡한 설치 없이 바로 텍스트를 입력해 브라우저 렌더링 결과를 시각적으로 확인한다.

## 실습 단계 / Steps
1. **StackBlitz 접속하기 / Access StackBlitz**
   - 브라우저에서 제공된 StackBlitz 링크를 클릭합니다.
2. **HTML 문서 살펴보기 / Inspect HTML Document**
   - 화면 왼쪽에 나타나는 텍스트 문서(`index.html`)를 관찰합니다.
3. **간단한 글자 입력 / Input Simple Text**
   - 뼈대 코드 사이에 본인의 이름이나 국가를 텍스트로 적어봅니다.
   - 예: `Hello, I am from Vietnam!`
4. **결과 확인 / Check the Result**
   - 오른쪽 뷰어(미리보기) 화면에 내가 친 글자가 즉각적으로 마법처럼 나타나는 것을 감상합니다.
