# 2주차 마이크로 과제 (Weekly Assignment)

## 과제 개요
본 과제는 수업 중 학습한 내용(환경 세팅 및 출력, 디버그)과 Agent 활용 능력을 결합하여 훈련하는 **마이크로 과제**입니다. 분량은 A4 영문 기준 절반(0.5페이지) 내외로 작성하여 eCampus에 제출합니다.

## 과제 주제: Agent를 활용한 '에러 메시지' 해독
코딩을 하다보면 긴 영어 에러 메시지와 맞닥뜨리게 됩니다. 일부러 코드를 망가뜨려서 복잡한 에러 메시지를 얻어낸 후, 이를 생성형 Agent에게 질문하여 원인을 파악해 봅니다.

## 과제 수행 단계

**Step 1. 에러 상황 만들기**
구글 Colab에서 다음 세 가지 중 하나의 실수를 고의로 저질러 에러를 발생시키세요.
1. `print`의 철자를 틀리게 작성하기 (예: `prnt("안녕")`)
2. 여는 따옴표와 닫는 따옴표의 종류를 다르게 작성하기 (예: `print("안녕')`)
3. 문장 중간에 한글과 영어 문장 부호가 섞여 들어간 경우 (예: 전각 괄호를 사용한 경우 등)
**발생한 에러 메시지의 전문을 텍스트로 복사해둡니다.**

**Step 2. Agent에게 질문 (Prompt)**
생성형 Agent (ChatGPT, Claude 등 자유 선택)에게 복사한 에러 메시지를 건네며 질의합니다.
* *작성 예시:* "파이썬에서 코드를 돌리다가 다음과 같은 에러가 났어. [복사한 에러 메시지] 왜 이런 오류가 난 것이고, 초보자가 이해하기 쉽게 코드를 어떻게 고쳐야 하는지 알려줘."

**Step 3. 과제 보고서 작성 필수 포함 항목**
아래 3가지 항목을 작성하여 워드, 한글 또는 PDF 형식으로 정리해 제출하세요.
1. **내 질문 (Agent Prompt):** 본인이 Agent에게 입력한 질문 내용 및 오류를 발생시킨 코드
2. **Agent 답변 비평 (Agent Critique):** Agent가 제시한 해결책을 읽고, "Agent가 에러의 핵심을 잘 짚어주었다", 혹은 "이런 용어는 내가 이해하기 어려웠다" 등 한두 문장으로 짧게 평가(비평)하기
3. **최종 도출 결과 (Final Result):** 본인의 논리로 완벽하게 오류를 수정한 최종 소스 코드와 성공적으로 문자가 출력된 결과 화면(캡처 또는 텍스트 복사)

* **제출처:** 학교 eCampus 2주차 과제 제출란
* **주의사항:** Agent의 답변 내용을 보고서에 길게 복사하여 붙여넣지 마세요. 본인이 느낀 점과 요약, 최종 코드만 작성하는 것이 핵심입니다.
