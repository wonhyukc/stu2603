## 1. 클라우드 개발 환경 접속 (Google Colab)
여러분의 PC에 복잡한 프로그램을 설치할 필요 없이, 브라우저만 있으면 사용할 수 있는 클라우드 IDE인 **Google Colab** 환경을 준비합니다.

### Step 1: 구글 코랩 접속 및 로그인
1. 웹 브라우저 (크롬 권장)에서 구글(`google.com`)에 접속하여 본인의 구글 계정으로 로그인합니다.
2. 주소창에 `colab.research.google.com`을 입력하여 구글 코랩에 접속합니다.

### Step 2: 새 파이썬 환경 열기
1. 코랩 메인 화면 팝업에서 왼쪽 아래에 있는 **[새 노트]** 버튼을 클릭합니다.
2. 조금 기다리면 코드를 입력할 수 있는 새로운 화면이 열립니다. 상단의 제목 창(예: `Untitled0.ipynb`)을 클릭하여 이름을 `helloworld.ipynb`로 변경해 봅시다.

## 2. 첫 번째 스크립트 실행 실습

### Task 1: 코드 입력하고 실행하기
1. 웹 브라우저 화면에 보이는 회색의 긴 네모 상자(코드 셀)를 클릭합니다.
2. 아래 코드를 입력합니다.
   ```python
   print("나의 첫 파이썬 코드입니다.")
   ```
3. 셀의 왼쪽 끝에 나타나는 **둥근 재생 버튼(`▶`)**을 누르거나, 키보드의 **`Shift + Enter`** 키를 누릅니다. (처음 실행할 때는 클라우드 서버에 연결하느라 몇 초 정도 걸릴 수 있습니다.)
4. 셀 바로 아래에 출력 결과가 정상적으로 나타나는지 확인합니다!

## 3. 에러 발생 유도 및 오류 디버깅 훈련 (Bug Hunting)

**🚨 완벽하게 동작하는 코드를 망가뜨려서 어떤 에러가 나는지 확인하는 것이 진짜 공부입니다.**

1. `print()`의 따옴표나 괄호를 하나씩 지우고 실행해보세요. 에러 메시지를 잘 보세요.
   * `SyntaxError: unterminated string literal` (문자열이 안 끝났어!) 등등
2. `print`를 `printf`나 `prnt`처럼 오타를 내고 실행해보세요.
   * `NameError: name 'prnt' is not defined` ('prnt'가 뭔지 정의되지 않았어!)
3. 파이썬은 대문자와 소문자를 엄격하게 구별합니다. `Print("안녕")`을 실행해보고 차이를 느껴보세요.

어떤 에러 메시지가 여러분이 자주 겪게 될 메시지인지 눈에 익혀두는 것이 핵심입니다!
