# 2주차 파이썬 실습 (Lab) 

## 실습 개요
* **주제:** 개발 환경 접속 및 기본 출력 함수 실행
* **목표:**
  1. Google Colab 환경에 접속하고 새로운 노트를 생성할 수 있다.
  2. `print()` 함수를 이용하여 문자열과 숫자를 정상적으로 출력할 수 있다.
  3. 고의로 오류를 발생시키고 에러 메시지를 읽는 훈련을 수행한다.
* **준비물:** 인터넷 브라우저 (Chrome 권장), 구글 계정

## 실습 1. 구글 코랩(Colab) 환경 구성하기
1. 웹 브라우저를 열고 `colab.research.google.com`에 접속합니다.
2. 구글 계정으로 로그인합니다.
3. 팝업창에서 **[새 노트]**를 클릭하여 빈 실습 환경을 엽니다.
4. 왼쪽 상단의 파일 이름(`Untitled0.ipynb`)을 클릭하여 `2주차_실습_학번.ipynb` 형태로 변경합니다.

## 실습 2. 나의 첫 파이썬 코드 실행하기
1. 화면 중앙에 있는 입력 셀에 다음 코드를 입력합니다.
   ```python
   print("나의 첫 파이썬 프로그램입니다!")
   ```
2. 셀 좌측의 실행(▶) 버튼을 누르거나 키보드 `Shift + Enter` 단축키를 눌러 셀을 실행합니다.
3. 셀 바로 밑에 작성한 문구가 정확히 출력되었는지 확인합니다.
4. 새로운 코딩 셀을 추가하고 숫자 연산을 테스트해 봅니다.
   ```python
   print(10 + 20)
   ```
   *문자와 달리 숫자는 따옴표를 붙이지 않아도 계산기처럼 동작하여 `30`이 출력됨을 확인합니다.*

## 실습 3. 에러 유발 및 메시지 확인 (디버깅 연습)
1. 새로운 셀을 추가하고 다음 코드를 고의로 괄호를 빼서 작성해 봅시다.
   ```python
   print"오류가 나야 합니다"
   ```
2. 실행 시 발생하는 `SyntaxError`를 확인합니다. 화살표(`^`)가 가리키는 위치를 살펴보세요.
3. 따옴표를 빼먹은 실수를 해봅니다.
   ```python
   print(안녕)
   ```
4. 실행 시 발생하는 `NameError`를 확인합니다. 왜 이 에러가 발생했는지 강사님이 설명해주신 개념(문자와 변수의 차이)을 떠올려보세요. 
5. 확인한 에러 메시지를 수정하여 올바르게 고친 후 정상 작동하게 만드세요.
